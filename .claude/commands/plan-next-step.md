# 要求

1. **指出当前最强模块**：基于 `learning-progress.md` 中的能力状态，说明哪些模块已形成基础能力

2. **指出当前短板**：哪些模块尚未系统展开，哪些模块虽有记录但能力层级较低

3. **给出未来 2-4 周学习任务**：
   - 优先推进哪些模块
   - 每天/每周应完成的具体动作
   - 每个动作对应的模块
   - 预计推进到的能力状态

4. **给出具体项目或实验建议**：
   - 适合当前水平的项目/实验
   - 需要哪些前置条件
   - 预期产出物
   - 涉及哪些能力模块

5. **考虑现实约束**：
   - 当前学期周次（参考 `plan/school-calendar.md`）
   - 课程安排（参考 `plan/timetable.md`）
   - 是否有考试、作业、实验等占用时间
   - 不要安排明显超出每周可用时间的任务量

6. **所有建议必须服务于 Edge AI / 嵌入式 AI 方向**

---

# 输出格式

## 1. 当前能力状态概览

### 已形成能力的模块
- 模块 X：……（当前最高能力状态）
- 模块 X：……

### 需要加强的模块
- 模块 X：……（当前状态，下一步目标）

### 尚未展开的模块
- 模块 X：……

## 2. 未来 2-4 周学习任务

按周列出具体任务，每个任务标注模块、预计时间、输出物。

## 3. 项目/实验建议

具体项目名称、涉及模块、前置条件、预期产出。

## 4. 注意事项与提醒

基于现实约束的调整建议。
