# 每日时评 — Daily Commentary

AI 时代，表达与思考能力愈发珍贵。每天 3 道时评题目，逼迫自己写下观点，训练独立思考与精准表达。

## 规则

1. **每日 7:17 AM** 自动生成当日题目文件夹，内含 3 篇题目
2. 每篇写 **300-800 字**，直接在题目 md 文件下方作答
3. 写完后在微信通知 Claude（说"打分"或"评分"），Claude 逐篇评分
4. 每周日做一次周回顾，横向对比本周所有文章

## 题目来源

**每天 3 道全新命题**，不重复、不循环题库。由 AI 根据当日热点、科技进展、社会议题原创命制。

题目覆盖 5 个方向，每天从不同方向组合中选题：

| 方向 | 说明 |
|------|------|
| 科技与社会 | AI、自动化、隐私、能源、太空探索等 |
| 教育反思 | 大学教育、自学方法、能力 vs 学历等 |
| 人文思考 | 意义感、幸福感、人际关系、代际差异等 |
| 时事评论 | 近期热点事件的分析与立场表达 |
| 哲理思辨 | 自由意志、道德判断、技术哲学等 |

`topic-bank.md` 为历史题目归档，用于避免近期重复，不作选题来源。

## 写作要求

- **必须有明确立场**：不要说"一方面...另一方面..."和稀泥
- **论据至少 2 条**：数据、案例、类比、推理均可
- **结尾收束**：不能写着写着就没了，必须有结论

## 评分标准

见 [scoring-rubric.md](scoring-rubric.md)，满分 50 分。
