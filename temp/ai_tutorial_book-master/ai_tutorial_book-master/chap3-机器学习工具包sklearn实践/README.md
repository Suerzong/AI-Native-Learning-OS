# ai_tutorial_book

#### 介绍
《人工智能入门实践》（肖波）图书，第3章所有代码

### 在本地计算机的练习

1. “chap3-机器学习工具包sklearn实践”的各个文件下载到本地计算机，在资源管理器中，进入第三章目录

2. 地址栏输入cmd，进入命令行窗口

3. 在命令行窗口输入：conda activate base，进入虚拟环境

    若提示conda命令找不到，而电脑已经安装过，则需要配置环境变量path，将conda的路径添加到path环境变量中。

4. 在命令行窗口输入：jupyter notebook，启动浏览器

5. 在浏览器选择对应的文件，执行对应的cell代码
