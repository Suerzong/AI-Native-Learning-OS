# 中国地形地貌数据集探索

通过python代码，将不同地形地貌在地图上展示。代码和数据集已经存储在本目录下。

## 实战环境准备

本实验建议在python 3.11及以上版本使用，否则有可能因版本过低导致无法运行。

本实验需要geopandas和matplotlib包，因此需要先安装：

pip install geopandas matplotlib --user

若安装不了，或安装太慢，可以尝试使用国内安装源，如：

pip install geopandas matplotlib --user -i https://mirrors.aliyun.com/pypi/simple/ 


## 运行代码

python go.py 

运行后，会出现相应的地貌展示图。例如：

![中国地形地貌图](a.png)

## 修改代码

若要显示所有的地貌，可注释掉代码中的第10行。

若要显示其他的地貌，可修改代码中的第10行。

若要修改图例信息显示位置或字体大小，可修改代码中的第12行。
