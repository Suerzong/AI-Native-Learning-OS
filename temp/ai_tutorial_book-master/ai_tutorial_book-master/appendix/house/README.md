# 基于SKLearn对加州房价数据集进行预测（包括线性回归和分类）实战

需要在python环境中提前安装jupyterlab包、scikit-learn包、matplotlib包等：

pip install jupyterlab scikit-learn matplotlib --user

若安装不了，或安装太慢，可以尝试使用国内安装源，如：

pip install jupyterlab scikit-learn matplotlib --user -i https://mirrors.aliyun.com/pypi/simple/ 

代码需要在jupyter notebook环境中打开运行。
