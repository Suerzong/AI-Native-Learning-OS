# tech-intel

每日科技情报报告目录。

## 目录结构

```
tech-intel/
  YYYY-MM-DD/
    tech-intel-YYYY-MM-DD.md   # 当日报告
  README.md
```

每天运行 `/tech-intel` 时，agent 会自动创建以日期命名的子文件夹并将报告保存在其中。

## 报告内容

每日科技情报报告覆盖：

1. 今日最重要的 3 条信息
2. AI / LLM 动态
3. Edge AI / 嵌入式 AI 动态
4. GitHub 开源项目
5. 论文与研究
6. 今日噪音信息
7. 今日行动建议
8. 可写入知识库的条目
