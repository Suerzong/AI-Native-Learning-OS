# LangChain 文档索引（llms.txt）

这是 LangChain 的完整文档索引，供 AI 编码助手和 LLM 使用，以发现和访问所有可用的文档页面。

## 概述

`llms.txt` 文件是一个标准格式的文档索引，允许 AI 助手在探索文档前发现所有可用页面。这对于以下场景非常有用：

  * AI 编码助手需要了解 LangChain 的完整 API
  * LLM 需要在回答问题前查找相关文档
  * 自动化工具需要生成代码示例

## 获取文档索引

完整文档索引可在以下地址获取：

    https://docs.langchain.com/llms.txt

## 使用方法

  1. 首先获取 `llms.txt` 文件了解所有可用页面
  2. 然后根据需要访问特定的文档页面
  3. 使用文档内容为 AI 助手提供上下文

## LangChain 文档 MCP 服务器

LangChain 还提供了文档 MCP 服务器，允许 AI 编码助手（如 Claude、VSCode 等）实时访问最新的 LangChain 文档。

安装 MCP 服务器以获得最新文档：

  * 访问 [LangChain Docs MCP](https://docs.langchain.com/use-these-docs) 了解如何连接
  * 支持 Claude、VSCode、Cursor 等 AI 编码工具
