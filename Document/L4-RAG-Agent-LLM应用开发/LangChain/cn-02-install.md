# 安装 LangChain

安装 LangChain 包：

    pip install -U langchain
    # 需要 Python 3.10+

LangChain 提供了对数百个 LLM 和数千个其他集成的支持。这些集成位于独立的提供商包中。

    # 安装 OpenAI 集成
    pip install -U langchain-openai

    # 安装 Anthropic 集成
    pip install -U langchain-anthropic

查看[集成标签页](https://docs.langchain.com/oss/python/integrations/providers/overview)获取完整的可用集成列表。

安装 LangChain 后，你可以按照[快速开始指南](https://docs.langchain.com/oss/python/langchain/quickstart)开始构建你自己的智能体和应用程序。
