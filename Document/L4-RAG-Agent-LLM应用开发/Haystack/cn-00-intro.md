# Haystack 简介

Haystack 是一个**开源 AI 框架**，用于构建面向生产的 **AI Agent（智能体）**、**强大的 RAG 应用**以及**可扩展的多模态搜索系统**。使用可复用的组件构建管道（Pipeline），每个组件负责特定任务。你可以自定义和扩展管道以满足你的需求。了解更多关于 Haystack 的信息及其工作原理。

欢迎来到 Haystack

如果你想跳过介绍，直接开始安装和创建搜索应用，请参阅[快速入门](</docs/get-started>)。

Haystack 是一个开源 AI 编排框架，可用于构建各种使用场景下基于大语言模型（LLM）的强大、面向生产的应用。无论你是创建自主智能体（Agent）、多模态应用还是可扩展的 RAG 系统，Haystack 都提供了相应的工具，帮助你轻松地将想法转化为生产应用。

Haystack 采用模块化设计，允许你组合来自 OpenAI、Google、Anthropic 以及 Hugging Face Transformers 等开源项目的最佳技术。

Haystack 的核心基础由组件（Components）和管道（Pipelines）组成，同时还包括文档存储（Document Stores）、智能体（Agents）、工具（Tools）以及丰富的集成。在 [Haystack 概念概览](</docs/concepts-overview>)中了解更多关于 Haystack 的概念。

在活跃的开发者社区支持下，Haystack 已经发展成为一个全面且用户友好的 LLM 开发框架。
