# 生成器（Generators）

生成器负责在你提供提示（prompt）后生成文本。它们针对每种 LLM 技术（OpenAI、本地、TGI 等）有特定的实现。

| 生成器 | 描述 | 流式支持 |
|--------|------|----------|
| [AmazonBedrockChatGenerator](</docs/amazonbedrockchatgenerator>) | 通过 Amazon Bedrock 服务使用模型进行聊天补全。 | 支持 |
| [AmazonBedrockGenerator](</docs/amazonbedrockgenerator>) | 通过 Amazon Bedrock 服务使用模型生成文本。 | 支持 |
| [AIMLAPIChatGenerator](</docs/aimllapichatgenerator>) | 通过 AIMLAPI 使用 AI 模型进行聊天补全。 | 支持 |
| [AnthropicChatGenerator](</docs/anthropicchatgenerator>) | 使用 Anthropic 大语言模型（LLM）进行聊天补全。 | 支持 |
| [AnthropicVertexChatGenerator](</docs/anthropicvertexchatgenerator>) | 使用 AnthropicVertex API 进行聊天补全。 | 支持 |
| [AnthropicGenerator](</docs/anthropicgenerator>) | 使用 Anthropic 大语言模型（LLM）进行文本补全。 | 支持 |
| [AzureOpenAIChatGenerator](</docs/azureopenaichatgenerator>) | 通过 Azure 服务使用 OpenAI 的 LLM 进行聊天补全。 | 支持 |
| [AzureOpenAIGenerator](</docs/azureopenaigenerator>) | 通过 Azure 服务使用 OpenAI 的 LLM 生成文本。 | 支持 |
| [AzureOpenAIResponsesChatGenerator](</docs/azureopenairesponseschatgenerator>) | 通过 Azure 服务使用 OpenAI 的 Responses API 进行聊天补全，支持推理模型。 | 支持 |
| [CohereChatGenerator](</docs/coherechatgenerator>) | 使用 Cohere 的 LLM 进行聊天补全。 | 支持 |
| [CohereGenerator](</docs/coheregenerator>) | 使用 Cohere API 查询 LLM。 | 支持 |
| [CometAPIChatGenerator](</docs/cometapichatgenerator>) | 通过 Comet API 使用 AI 模型进行聊天补全。 | 支持 |
| [DALLEImageGenerator](</docs/dalleimagegenerator>) | 使用 OpenAI 的 DALL-E 模型生成图像。 | 不支持 |
| [FallbackChatGenerator](</docs/fallbackchatgenerator>) | ChatGenerator 包装器，按顺序尝试多个 ChatGenerator，直到有一个成功。 | 支持 |
| [GoogleAIGeminiChatGenerator](</docs/googleaigeminichatgenerator>) | 使用 Google Gemini 模型进行聊天补全。**此集成即将弃用，建议使用 [GoogleGenAIChatGenerator](</docs/googlegenaichatgenerator>)。** | 支持 |
| [GoogleAIGeminiGenerator](</docs/googleaigeminigenerator>) | 使用 Google Gemini 模型生成文本。**此集成即将弃用，建议使用 [GoogleGenAIChatGenerator](</docs/googlegenaichatgenerator>)。** | 支持 |
| [GoogleGenAIChatGenerator](</docs/googlegenaichatgenerator>) | 通过 Google Gen AI SDK 使用 Google Gemini 模型进行聊天补全。 | 支持 |
| [HuggingFaceAPIChatGenerator](</docs/huggingfaceapichatgenerator>) | 使用各种 Hugging Face API 进行聊天补全。 | 支持 |
| [HuggingFaceAPIGenerator](</docs/huggingfaceapigenerator>) | 使用各种 Hugging Face API 生成文本。 | 支持 |
| [HuggingFaceLocalChatGenerator](</docs/huggingfacelocalchatgenerator>) | 使用本地运行的 Hugging Face 模型进行聊天补全。 | 支持 |
| [HuggingFaceLocalGenerator](</docs/huggingfacelocalgenerator>) | 使用本地运行的 Hugging Face 模型生成文本。 | 支持 |
| [LlamaCppChatGenerator](</docs/llamacppchatgenerator>) | 使用在 Llama.cpp 上运行的 LLM 进行聊天补全。 | 不支持 |
| [LlamaCppGenerator](</docs/llamacppgenerator>) | 使用在 Llama.cpp 上运行的 LLM 生成文本。 | 不支持 |
| [LlamaStackChatGenerator](</docs/llamastackchatgenerator>) | 使用通过 Llama Stack 服务器提供的 LLM 模型进行聊天补全。 | 支持 |
| [MetaLlamaChatGenerator](</docs/metallamachatgenerator>) | 使用 Meta Llama API 上托管的任何模型进行聊天补全。 | 支持 |
| [MistralChatGenerator](</docs/mistralchatgenerator>) | 使用 Mistral 的文本生成模型进行聊天补全。 | 支持 |
| [NvidiaChatGenerator](</docs/nvidiachatgenerator>) | 使用 Nvidia 托管的模型进行聊天补全。 | 支持 |
| [NvidiaGenerator](</docs/nvidiagenerator>) | 使用 NVIDIA NIM 自托管的 LLM 或 NVIDIA API 目录上托管的模型提供文本生成接口。 | 不支持 |
| [OllamaChatGenerator](</docs/ollamachatgenerator>) | 使用在 Ollama 上运行的 LLM 进行聊天补全。 | 支持 |
| [OllamaGenerator](</docs/ollamagenerator>) | 使用在 Ollama 上运行的 LLM 提供文本生成接口。 | 支持 |
| [OpenAIChatGenerator](</docs/openaichatgenerator>) | 使用 OpenAI 的大语言模型（LLM）进行聊天补全。 | 支持 |
| [OpenAIGenerator](</docs/openaigenerator>) | 使用 OpenAI 的大语言模型（LLM）生成文本。 | 支持 |
| [OpenAIResponsesChatGenerator](</docs/openairesponseschatgenerator>) | 使用 OpenAI 的 Responses API 进行聊天补全，支持推理模型。 | 支持 |
| [OpenRouterChatGenerator](</docs/openrouterchatgenerator>) | 使用 OpenRouter 上托管的任何模型进行聊天补全。 | 支持 |
| [SagemakerGenerator](</docs/sagemakergenerator>) | 使用部署在 Amazon Sagemaker 上的 LLM 生成文本。 | 不支持 |
| [STACKITChatGenerator](</docs/stackitchatgenerator>) | 使用 STACKIT API 进行聊天补全。 | 支持 |
| [TogetherAIChatGenerator](</docs/togetheraichatgenerator>) | 使用 Together AI 上托管的模型进行聊天补全。 | 支持 |
| [TogetherAIGenerator](</docs/togetheraigenerator>) | 使用 Together AI 上托管的模型生成文本。 | 支持 |
| [VertexAICodeGenerator](</docs/vertexaicodegenerator>) | 使用 Google Vertex AI 生成模型生成代码。 | 不支持 |
| [VertexAIGeminiChatGenerator](</docs/vertexaigeminichatgenerator>) | 通过 GCP Vertex AI 使用 Google Gemini 模型进行聊天补全。**此集成即将弃用，建议使用 [GoogleGenAIChatGenerator](</docs/googlegenaichatgenerator>)。** | 支持 |
| [VertexAIGeminiGenerator](</docs/vertexaigeminigenerator>) | 通过 GCP Vertex AI 使用 Google Gemini 模型生成文本。**此集成即将弃用，建议使用 [GoogleGenAIChatGenerator](</docs/googlegenaichatgenerator>)。** | 支持 |
| [VertexAIImageCaptioner](</docs/vertexaiimagecaptioner>) | 使用 Google Vertex AI `imagetext` 生成模型生成文本。 | 不支持 |
| [VertexAIImageGenerator](</docs/vertexaiimagegenerator>) | 使用 Google Vertex AI 生成模型生成图像。 | 不支持 |
| [VertexAIImageQA](</docs/vertexaiimageqa>) | 使用 Google Vertex AI 生成模型进行文本生成（图像描述）。 | 不支持 |
| [VertexAITextGenerator](</docs/vertexaitextgenerator>) | 使用 Google Vertex AI 生成模型生成文本。 | 不支持 |
| [VLLMChatGenerator](</docs/vllmchatgenerator>) | 使用 vLLM 提供的模型进行聊天补全。 | 支持 |
| [WatsonxGenerator](</docs/watsonxgenerator>) | 使用 IBM Watsonx 模型生成文本。 | 支持 |
| [WatsonxChatGenerator](</docs/watsonxchatgenerator>) | 使用 IBM Watsonx 模型进行聊天补全。 | 支持 |
