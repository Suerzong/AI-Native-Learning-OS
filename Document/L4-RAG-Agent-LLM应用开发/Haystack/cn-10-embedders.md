# 嵌入器（Embedders）

Haystack 中的嵌入器使用预训练模型将文本或文档转换为向量表示。然后你可以将嵌入用于问答、信息检索等任务。

提示：有关如何选择合适嵌入器的一般指导，请阅读我们的[选择合适的嵌入器](</docs/choosing-the-right-embedder>)页面。

以下是 Haystack 中可用的嵌入器：

| 嵌入器 | 描述 |
|--------|------|
| [AmazonBedrockTextEmbedder](</docs/amazonbedrocktextembedder>) | 通过 Amazon Bedrock API 使用模型计算文本（如查询）的嵌入。 |
| [AmazonBedrockDocumentEmbedder](</docs/amazonbedrockdocumentembedder>) | 通过 Amazon Bedrock API 使用模型计算文档的嵌入。 |
| [AmazonBedrockDocumentImageEmbedder](</docs/amazonbedrockdocumentimageembedder>) | 计算文档的图像嵌入。 |
| [AzureOpenAITextEmbedder](</docs/azureopenaitextembedder>) | 通过 Azure 部署的 OpenAI 模型计算文本（如查询）的嵌入。 |
| [AzureOpenAIDocumentEmbedder](</docs/azureopenaidocumentembedder>) | 通过 Azure 部署的 OpenAI 模型计算文档的嵌入。 |
| [CohereTextEmbedder](</docs/coheretextembedder>) | 使用 Cohere 模型嵌入简单字符串（如查询）。需要 Cohere API 密钥。 |
| [CohereDocumentEmbedder](</docs/coheredocumentembedder>) | 使用 Cohere 模型嵌入文档列表。需要 Cohere API 密钥。 |
| [CohereDocumentImageEmbedder](</docs/coheredocumentimageembedder>) | 计算文档列表的图像嵌入，并将获得的向量存储在每个文档的嵌入字段中。 |
| [FastembedTextEmbedder](</docs/fastembedtextembedder>) | 使用 Fastembed 支持的嵌入模型计算字符串的嵌入。 |
| [FastembedDocumentEmbedder](</docs/fastembeddocumentembedder>) | 使用 Fastembed 支持的模型计算文档列表的嵌入。 |
| [FastembedSparseTextEmbedder](</docs/fastembedsparsetextembedder>) | 使用 Fastembed 支持的模型将简单字符串（如查询）嵌入为稀疏向量。 |
| [FastembedSparseDocumentEmbedder](</docs/fastembedsparsedocumentembedder>) | 使用 Fastembed 支持的模型为文档列表添加稀疏嵌入。 |
| [GoogleGenAITextEmbedder](</docs/googlegenaitextembedder>) | 使用 Google AI 模型嵌入简单字符串（如查询）。需要 Google API 密钥。 |
| [GoogleGenAIDocumentEmbedder](</docs/googlegenaidocumentembedder>) | 使用 Google AI 模型嵌入文档列表。需要 Google API 密钥。 |
| [GoogleGenAIMultimodalDocumentEmbedder](</docs/googlegenaimultimodaldocumentembedder>) | 使用 Google AI 模型嵌入非文本文档列表。需要 Google API 密钥。 |
| [HuggingFaceAPIDocumentEmbedder](</docs/huggingfaceapidocumentembedder>) | 使用各种 Hugging Face API 计算文档嵌入。 |
| [HuggingFaceAPITextEmbedder](</docs/huggingfaceapitextembedder>) | 使用各种 Hugging Face API 嵌入字符串。 |
| [JinaTextEmbedder](</docs/jinatextembedder>) | 使用 Jina AI 嵌入模型嵌入简单字符串（如查询）。需要 Jina AI API 密钥。 |
| [JinaDocumentEmbedder](</docs/jinadocumentembedder>) | 使用 Jina AI 嵌入模型嵌入文档列表。需要 Jina AI API 密钥。 |
| [JinaDocumentImageEmbedder](</docs/jinadocumentimageembedder>) | 计算文档列表的图像嵌入，并将获得的向量存储在每个文档的嵌入字段中。 |
| [MistralTextEmbedder](</docs/mistraltextembedder>) | 使用 Mistral API 和模型将字符串转换为向量。 |
| [MistralDocumentEmbedder](</docs/mistraldocumentembedder>) | 使用 Mistral API 和模型计算文档列表的嵌入。 |
| [NvidiaTextEmbedder](</docs/nvidiatextembedder>) | 将简单字符串（如查询）嵌入为向量。 |
| [NvidiaDocumentEmbedder](</docs/nvidiadocumentembedder>) | 使用其内容的嵌入丰富文档的元数据。 |
| [OllamaTextEmbedder](</docs/ollamatextembedder>) | 使用与 Ollama Library 兼容的嵌入模型计算字符串的嵌入。 |
| [OllamaDocumentEmbedder](</docs/ollamadocumentembedder>) | 使用与 Ollama Library 兼容的嵌入模型计算文档列表的嵌入。 |
| [OpenAIDocumentEmbedder](</docs/openaidocumentembedder>) | 使用 OpenAI 嵌入模型嵌入文档列表。需要有效 OpenAI 账户的 API 密钥。 |
| [OpenAITextEmbedder](</docs/openaitextembedder>) | 使用 OpenAI 嵌入模型嵌入简单字符串（如查询）。需要有效 OpenAI 账户的 API 密钥。 |
| [OptimumTextEmbedder](</docs/optimumtextembedder>) | 使用 Hugging Face Optimum 库加载的模型嵌入文本。 |
| [OptimumDocumentEmbedder](</docs/optimumdocumentembedder>) | 使用 Hugging Face Optimum 库加载的模型计算文档的嵌入。 |
| [SentenceTransformersTextEmbedder](</docs/sentencetransformerstextembedder>) | 使用 Sentence Transformer 模型嵌入简单字符串（如查询）。 |
| [SentenceTransformersDocumentEmbedder](</docs/sentencetransformersdocumentembedder>) | 使用 Sentence Transformer 模型嵌入文档列表。 |
| [SentenceTransformersDocumentImageEmbedder](</docs/sentencetransformersdocumentimageembedder>) | 计算文档列表的图像嵌入，并将获得的向量存储在每个文档的嵌入字段中。 |
| [SentenceTransformersSparseTextEmbedder](</docs/sentencetransformerssparsetextembedder>) | 使用 Sentence Transformers 模型将简单字符串（如查询）嵌入为稀疏向量。 |
| [SentenceTransformersSparseDocumentEmbedder](</docs/sentencetransformerssparsedocumentembedder>) | 使用 Sentence Transformers 模型为文档列表添加稀疏嵌入。 |
| [STACKITTextEmbedder](</docs/stackittextembedder>) | 使用 STACKIT API 进行文本嵌入。 |
| [STACKITDocumentEmbedder](</docs/stackitdocumentembedder>) | 使用 STACKIT API 进行文档嵌入。 |
| [VertexAITextEmbedder](</docs/vertexaitextembedder>) | 通过 VertexAI Embeddings API 使用模型计算文本（如查询）的嵌入。**此集成即将弃用，建议使用 [GoogleGenAITextEmbedder](</docs/googlegenaitextembedder>)。** |
| [VertexAIDocumentEmbedder](</docs/vertexaidocumentembedder>) | 通过 VertexAI Embeddings API 使用模型计算文档的嵌入。**此集成即将弃用，建议使用 [GoogleGenAIDocumentEmbedder](</docs/googlegenaidocumentembedder>)。** |
| [VLLMTextEmbedder](</docs/vllmtextembedder>) | 使用 vLLM 提供的模型计算字符串的嵌入。 |
| [VLLMDocumentEmbedder](</docs/vllmdocumentembedder>) | 使用 vLLM 提供的模型计算文档列表的嵌入。 |
| [WatsonxTextEmbedder](</docs/watsonxtextembedder>) | 使用 IBM Watsonx 模型计算文本（如查询）的嵌入。 |
| [WatsonxDocumentEmbedder](</docs/watsonxdocumentembedder>) | 使用 IBM Watsonx 模型计算文档的嵌入。 |
