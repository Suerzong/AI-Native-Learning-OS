# Claude 简介

复制页面

Claude 是由 Anthropic 构建的高性能、可信赖且智能的 AI（人工智能）平台。Claude 擅长处理涉及语言、推理、分析、编程等任务。

复制页面

最新一代 Claude 模型：

**Claude Opus 4.7** - 我们最强大的模型，适用于复杂推理和智能体编程，相比 Claude Opus 4.6 有质的飞跃。[了解更多](<https://www.anthropic.com/news/claude-opus-4-7>)。

**Claude Sonnet 4.6** - 大规模前沿智能——专为编程、智能体和企业工作流构建。[了解更多](<https://www.anthropic.com/news/claude-sonnet-4-6>)。

**Claude Haiku 4.5** - 速度最快且接近前沿智能的模型。[了解更多](<https://www.anthropic.com/news/claude-haiku-4-5>)。

想与 Claude 对话？访问 [claude.ai](<https://www.claude.ai>)。

Anthropic 提供两种使用 Claude 的方式，分别适用于不同场景：

| Messages API | Claude 托管智能体
---|---|---
**功能** | 直接模型提示访问 | 预构建、可配置的智能体框架，运行在托管基础设施上
**适用场景** | 自定义智能体循环和细粒度控制 | 长时间运行的任务和异步工作
**了解更多** | [Messages API 文档](</docs/en/build-with-claude/working-with-messages>) | [Claude 托管智能体文档](</docs/en/managed-agents/overview>)

##

新开发者推荐路径

按照以下步骤，从零开始完成 Claude 集成。

  1. 1

发起第一次 API 调用

设置环境、安装 SDK，并向 Claude 发送第一条消息。

[前往快速入门](</docs/en/get-started>)

  2. 2

理解 Messages API

学习核心请求和响应结构，包括多轮对话、系统提示和停止原因。

[阅读 Messages API 指南](</docs/en/build-with-claude/working-with-messages>)

  3. 3

选择合适的模型

按能力和成本对比 Claude 模型，选择最适合你场景的模型。

[查看模型概览](</docs/en/about-claude/models/overview>)

  4. 4

探索功能和工具

了解 Claude 能做什么：扩展思考、网络搜索、文件处理、结构化输出等。

[浏览功能概览](</docs/en/build-with-claude/overview>)
