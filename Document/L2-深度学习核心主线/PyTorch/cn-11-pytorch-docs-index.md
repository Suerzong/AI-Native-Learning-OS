## 文档

PyTorch 是一个使用 GPU 和 CPU 进行深度学习的优化张量库。

PyTorch 的主要特性包括：

  * 类似于 NumPy 的张量计算，但可以在 GPU 上运行
  * 基于磁带的自动微分系统
  * 支持动态神经网络

PyTorch 提供了两个主要功能：

  * 一个 n 维张量，类似于 NumPy 但可以在 GPU 上运行
  * 用于构建和训练神经网络的自动微分
