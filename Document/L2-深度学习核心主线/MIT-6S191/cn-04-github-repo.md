# 说明

[](#instructions)

MIT 深度学习导论软件实验旨在按你自己的节奏完成。在每个实验结束时，将有说明如何提交你的材料作为实验竞赛的一部分。这些说明包括必须提交哪些信息以及以什么格式提交。

## 在 Google Colaboratory 中打开实验：

[](#opening-the-labs-in-google-colaboratory)

2026 年深度学习导论实验将在 Google 的 Colaboratory 中运行，这是一个完全在云端运行的 Jupyter 笔记本环境，因此你无需下载任何内容。要运行这些实验，你必须拥有 Google 帐户。

在这个 Github 仓库中，导航到你想运行的实验文件夹（`lab1`、`lab2`、`lab3`）并打开相应的 Python 笔记本（*.ipynb）。点击实验顶部的"在 Colab 中运行"链接。就是这样！

## 运行实验

[](#running-the-labs)

现在，要在 Colab 上运行实验，请打开 Jupyter 笔记本。导航到"运行时"选项卡 --> "更改运行时类型"。在弹出窗口中，"运行时类型"下选择"Python 3"，"硬件加速器"下选择"GPU"。浏览笔记本并填写 `#TODO` 单元格以使代码为你编译！

### MIT 深度学习包

[](#mit-deep-learning-package)

你可能会注意到在实验中我们从 Python 包仓库安装了 `mitdeeplearning` Python 包：

`pip install mitdeeplearning`

该包含有我们在整个课程中使用的便捷函数，可以像任何其他 Python 包一样导入。

`>>> import mitdeeplearning as mdl`

我们在每个实验中都为你做了这个，但该包也在相同的许可证下开源，因此你也可以在课外使用它。
