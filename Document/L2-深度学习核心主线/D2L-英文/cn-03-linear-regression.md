# 3. 用于回归的线性神经网络

在我们担心使神经网络变深之前，先实现一些浅层网络会很有帮助，其中输入直接连接到输出。这很重要，有几个原因。首先，与其被复杂的架构分心，我们可以专注于神经网络训练的基础，包括参数化输出层、处理数据、指定损失函数和训练模型。其次，这类浅层网络恰好构成了一组线性模型，其中包括许多经典的统计预测方法，包括线性回归和 softmax 回归。理解这些经典工具至关重要，因为它们在许多场景中被广泛使用，我们经常需要将它们用作基线来证明使用更复杂架构的合理性。本章将专注于线性回归，下一章将通过开发用于分类的线性神经网络来扩展我们的建模能力。
