# 6. 构建指南

除了庞大的数据集和强大的硬件之外，优秀的软件工具在深度学习的快速发展中发挥了不可或缺的作用。从 2007 年发布的开创性 Theano 库开始，灵活的开源工具使研究人员能够快速构建模型原型，在回收标准组件时避免重复工作，同时仍然保持进行底层修改的能力。随着时间的推移，深度学习的库已经发展到提供越来越粗粒度的抽象。正如半导体设计师从指定晶体管到逻辑电路再到编写代码一样，神经网络研究人员已经从思考单个人工神经元的行为转变为以整个层的概念来构思网络，现在通常设计的是以更粗粒度的_块_为思路的架构。

到目前为止，我们已经介绍了一些基本的机器学习概念，逐步过渡到功能齐全的深度学习模型。在上一章中，我们从头实现了 MLP 的每个组件，甚至展示了如何利用高级 API 轻松推出相同的模型。为了让你快速达到那个水平，我们_调用了_库，但跳过了关于_它们如何工作_的更高级细节。在本章中，我们将揭开帷幕，更深入地研究深度学习计算的关键组件，即模型构造、参数访问和初始化、设计自定义层和块、将模型读写到磁盘，以及利用 GPU 实现显著的加速。这些洞察将使你从_终端用户_转变为_高级用户_，为你提供在保留实现更复杂模型（包括你自己发明的模型）灵活性的同时，从成熟的深度学习库中获益所需的工具。虽然本章没有引入任何新的模型或数据集，但后续的高级建模章节在很大程度上依赖于这些技术。
