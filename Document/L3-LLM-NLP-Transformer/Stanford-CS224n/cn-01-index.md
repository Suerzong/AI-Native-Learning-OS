# CS224N：深度学习自然语言处理

### 斯坦福 / 2026 年冬季学期

自然语言处理（NLP）是人工智能（AI）的关键组成部分，研究人们如何共享信息。近年来，深度学习方法在许多 NLP 任务上取得了非常高的性能。在本课程中，学生将全面了解用于 NLP 的前沿神经网络。
