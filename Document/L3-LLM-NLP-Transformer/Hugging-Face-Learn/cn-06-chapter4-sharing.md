# [](#the-hugging-face-hub) Hugging Face Hub

[](https://discuss.huggingface.co/t/chapter-4-questions)

[Hugging Face Hub](https://huggingface.co/) —— 我们的主网站 —— 是一个核心平台，使任何人都可以发现、使用和贡献新的最先进模型和数据集。它托管了各种各样的模型，有超过 10,000 个公开可用。本章我们将专注于模型，并在第 5 章中了解数据集。

Hub 中的模型不仅限于 🤗 Transformers，甚至不仅限于 NLP。有来自 [Flair](https://github.com/flairNLP/flair) 和 [AllenNLP](https://github.com/allenai/allennlp) 的 NLP 模型、来自 [Asteroid](https://github.com/asteroid-team/asteroid) 和 [pyannote](https://github.com/pyannote/pyannote-audio) 的语音模型，以及来自 [timm](https://github.com/rwightman/pytorch-image-models) 的视觉模型等等。

每个模型都作为 Git 仓库托管，支持版本控制和可复现性。在 Hub 上共享模型意味着向社区开放，使任何希望轻松使用它的人都能访问，从而消除了自行训练模型的需求，并简化了共享和使用。

此外，在 Hub 上共享模型会自动为该模型部署一个托管的 Inference API。社区中的任何人都可以在模型页面上直接使用自定义输入和合适的组件进行测试。

最棒的是，在 Hub 上共享和使用任何公共模型完全免费！如果你想私密共享模型，也有[付费方案](https://huggingface.co/pricing)可供选择。
