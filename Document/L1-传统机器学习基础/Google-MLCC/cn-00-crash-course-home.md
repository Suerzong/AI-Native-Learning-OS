# 机器学习速成课程

自 2018 年以来，全球数百万人依靠机器学习速成课程（Machine Learning Crash Course，MLCC）来学习机器学习的工作原理，以及机器学习如何为他们所用。我们很高兴宣布推出了更新版 MLCC，涵盖了 AI 领域的最新进展，并更加注重交互式学习。观看此视频以了解更多关于新版 MLCC 的信息。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Linear_Regression_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/linear-regression>)

线性回归（Linear Regression）入门，涵盖线性模型、损失函数、梯度下降（Gradient Descent）和超参数调优（Hyperparameter Tuning）。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Logistic_Regression_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/logistic-regression>)

逻辑回归（Logistic Regression）入门，其中 ML 模型用于预测给定结果的概率。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Classification_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/classification>)

二元分类（Binary Classification）模型入门，涵盖阈值判定、混淆矩阵（Confusion Matrix），以及准确率（Accuracy）、精确率（Precision）、召回率（Recall）和 AUC 等指标。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Numerical_Data_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/numerical-data>)

学习如何分析和转换数值数据（Numerical Data），以更有效地训练 ML 模型。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Categorical_Data_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/categorical-data>)

学习处理类别数据（Categorical Data）的基础知识：如何区分类别数据与数值数据；如何使用独热编码（One-Hot Encoding）、特征哈希（Feature Hashing）和均值编码（Mean Encoding）将类别数据数值化；以及如何进行特征交叉（Feature Cross）。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Generalization_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/overfitting>)

机器学习数据集特征入门，以及如何准备数据以确保训练和评估模型时获得高质量结果。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Neural_Nets_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/neural-networks>)

神经网络（Neural Network）架构基本原理入门，包括感知机（Perceptron）、隐藏层（Hidden Layer）和激活函数（Activation Function）。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Embedding_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/embeddings>)

学习嵌入（Embedding）如何让你对大型特征向量进行机器学习。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Large_Lanuage_Model_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/llm>)

大语言模型（Large Language Model，LLM）入门，从词元（Token）到 Transformer。学习 LLM 如何学习预测文本输出的基础知识，以及它们的架构和训练方式。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Production_ML_Systems_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/production-ml-systems>)

学习机器学习生产系统如何跨多个组件协同工作。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Auto_ML_Yellow_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/automl>)

学习使用自动化机器学习（AutoML）的原则和最佳实践。

[ ![](https://developers.google.com/static/machine-learning/crash-course/images/MLCC_Fairness_no_text.png) ](<https://developers.google.com/machine-learning/crash-course/fairness>)

学习审计 ML 模型公平性（Fairness）的原则和最佳实践，包括识别和缓解数据中偏见（Bias）的策略。
