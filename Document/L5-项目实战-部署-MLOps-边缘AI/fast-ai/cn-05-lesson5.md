# 5：从头开始的模型

今天，我们研究如何使用 Python 和 PyTorch 从头开始​​创建神经网络，以及如何实现训练循环来优化模型的权重。我们从单层回归模型构建到具有一个隐藏层的神经网络，然后构建深度学习模型。在此过程中，我们还将了解如何使用名为 _sigmoid_ 的特殊函数来使二元分类模型更易于训练，并且我们还将了解 _metrics_。
