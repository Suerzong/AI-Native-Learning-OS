# 2: 部署

今天你将设计自己的机器学习项目，创建自己的数据集，使用你的数据训练模型，最后在 Web 上部署应用程序。我们将使用一个特定的部署目标——[Hugging Face Space](https://huggingface.co/spaces) 和 [Gradio](https://gradio.app/)，并了解如何使用 JavaScript 在浏览器中实现界面。部署到其他服务的方法与你在本课中学习的方法非常相似。
