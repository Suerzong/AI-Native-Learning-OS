# MLflow 文档

欢迎来到 MLflow 文档。我们的文档分为两个部分，帮助你快速找到所需内容。选择 [**大语言模型与智能体**](/docs/latest/genai/) 了解智能体和 LLM 应用开发、追踪和评估工具；选择 [**机器学习**](/docs/latest/ml/) 了解传统 ML 工作流。

## 大语言模型与智能体

探索 LLM 和智能体的可观测性工具、提示词管理、基础模型部署和评估框架。学习如何使用 MLflow 追踪、评估和优化你的 LLM 应用和智能体工作流。

[开源版本 →](/docs/latest/genai/)[Databricks 上的 MLflow →](https://docs.databricks.com/aws/en/mlflow3/genai/)

## 机器学习

获取实验跟踪、模型打包、注册管理和部署的综合指南。了解 MLflow 在传统机器学习工作流、超参数调优和模型生命周期管理方面的核心功能。

[开源版本 →](/docs/latest/ml/)[Databricks 上的 MLflow →](https://docs.databricks.com/aws/en/mlflow/)
