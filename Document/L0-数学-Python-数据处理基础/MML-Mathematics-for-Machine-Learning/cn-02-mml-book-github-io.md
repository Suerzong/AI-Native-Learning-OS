# mml-book.github.io

[](#mml-bookgithubio)

《机器学习的数学》一书的配套网页

<https://mml-book.com>

版权所有 2020 Marc Peter Deisenroth、A Aldo Faisal 和 Cheng Soon Ong。将由剑桥大学出版社出版。

我们正在编写一本关于机器学习数学的书籍，旨在激励人们学习数学概念。本书并不打算涵盖高级机器学习技术，因为已有大量书籍在做这件事。相反，我们的目标是提供阅读那些书籍所需的数学技能。

我们将本书分为两部分：

- 数学基础
- 使用数学基础的机器学习算法示例

我们力求本书篇幅适中，因此无法涵盖所有内容。我们还将为第一部分提供练习题，为第二部分提供 Jupyter Notebook。

Notebook 可以在 [![Binder](https://camo.githubusercontent.com/c351dded6463f29bc0945aaaa25a1ab9556d5f7e02c99d2c74dd641a5eb7dee9/68747470733a2f2f6d7962696e6465722e6f72672f62616467655f6c6f676e6f732e737667)](https://mybinder.org/v2/gh/mml-book/mml-book.github.io/master?filepath=tutorials) 上在线运行。
