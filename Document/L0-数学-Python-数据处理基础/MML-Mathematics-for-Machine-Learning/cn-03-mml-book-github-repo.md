![书籍封面](/static/images/mml-book-cover.jpg)

我们编写了一本关于机器学习数学的书籍，旨在激励人们学习数学概念。本书并不打算涵盖高级机器学习技术，因为已有大量书籍在做这件事。相反，我们的目标是提供阅读那些书籍所需的数学技能。

本书可在[剑桥大学出版社出版](https://www.cambridge.org/academic/subjects/computer-science/pattern-recognition-and-machine-learning/mathematics-machine-learning?format=PB)购买（2020 年 4 月出版）。

我们将本书分为两部分：

- 数学基础
- 使用数学基础的机器学习算法示例

我们力求本书篇幅适中，因此并未涵盖所有内容。

**我们将持续提供本书的[免费 PDF 版本](/book/mml-book.pdf)。**

## 目录

**第一部分：数学基础**

1. **引言与动机**
2. **线性代数（Linear Algebra）**
3. **解析几何（Analytic Geometry）**
4. **矩阵分解（Matrix Decompositions）**
5. **向量微积分（Vector Calculus）**
6. **概率与分布（Probability and Distribution）**
7. **连续优化（Continuous Optimization）**

**第二部分：核心机器学习问题**

8. **当模型遇到数据**
9. **线性回归（Linear Regression）**
10. **主成分分析降维（Dimensionality Reduction with PCA）**
11. **高斯混合模型密度估计（Density Estimation with GMM）**
12. **支持向量机分类（Classification with SVM）**
