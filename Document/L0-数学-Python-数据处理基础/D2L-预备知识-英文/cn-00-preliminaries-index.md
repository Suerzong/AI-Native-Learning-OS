# 2\. 预备知识

在深入学习之前，你需要掌握一些基本技能：（i）存储和操作数据的技术；（ii）从各种来源摄取和预处理数据的库；（iii）应用于高维数据元素的基本线性代数运算知识；（iv）足够的微积分知识，以便确定调整每个参数的方向以降低损失函数；（v）自动计算导数的能力，这样你就可以忘掉大部分刚学的微积分知识；（vi）对概率的基本掌握，这是我们进行不确定性推理的主要语言；（vii）在遇到困难时从官方文档中查找答案的能力。
