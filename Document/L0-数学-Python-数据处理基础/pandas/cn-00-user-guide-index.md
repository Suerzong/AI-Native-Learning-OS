# 用户指南[#](#user-guide "Link to this heading")

用户指南按主题领域全面覆盖 pandas 的所有功能。每个小节介绍一个主题（例如"缺失数据处理"），并讨论 pandas 如何解决该问题，贯穿大量示例。

pandas 的全新用户应从 [10 分钟入门 pandas](10min.html#min) 开始。

关于 pandas 基础的高级概述，请参阅[数据结构简介](dsintro.html#dsintro)和[基本功能](basics.html#basics)。

任何具体方法的更多信息可在 [API 参考](../reference/index.html#api) 中获取。

## 如何阅读这些指南[#](#how-to-read-these-guides "Link to this heading")

在这些指南中，你会看到代码块中的输入代码，例如：

    import pandas as pd
    pd.DataFrame({'A': [1, 2, 3]})

或者：

    In [1]: import pandas as pd

    In [2]: pd.DataFrame({'A': [1, 2, 3]})
    Out[2]:
       A
    0  1
    1  2
    2  3

第一个代码块是标准的 Python 输入，而第二个代码块中 `In [1]:` 表示输入位于 [Jupyter Notebook](https://jupyter.org) 中。在 Jupyter Notebook 中，最后一行会被打印，图表会内联显示。

例如：

    In [3]: a = 1

    In [4]: a
    Out[4]: 1

等价于：

    a = 1
    print(a)
