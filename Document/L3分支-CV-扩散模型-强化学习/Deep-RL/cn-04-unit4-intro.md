# Deep RL Course

## Unity ML-Agents 简介

强化学习的挑战之一是**创建环境**。幸运的是，我们可以使用游戏引擎来做到这一点。这些引擎，如 Unity、Godot 或 Unreal Engine，是为创建视频游戏而设计的程序。它们非常适合创建环境：提供物理系统、2D/3D 渲染等。

其中，Unity 创建了 Unity ML-Agents Toolkit，一个基于 Unity 游戏引擎的插件，允许我们**使用 Unity 游戏引擎作为环境构建器来训练智能体**。

Unity ML-Agents Toolkit 提供了许多出色的预制环境，从踢足球（足球）、学习行走到跳过大墙。

在本单元中，我们将学习使用 ML-Agents，**即使你不知道如何使用 Unity 游戏引擎也不用担心**：你不需要使用它来训练智能体。

今天，我们将训练两个智能体：

  * 第一个将学习**向生成的目标投掷雪球**。
  * 第二个需要**按下按钮生成金字塔，然后导航到金字塔，将其推倒，并移动到顶部的金砖**。为此，它需要探索环境，这将使用一种称为好奇心（Curiosity）的技术来完成。

训练后，**你将把训练好的智能体推送到 Hugging Face Hub**，你将能够**直接在浏览器中可视化它们的行为，无需使用 Unity 编辑器**。

完成本单元将**为你准备下一个挑战：AI 对战，你将在多智能体环境中训练智能体并与同学的智能体竞争**。

让我们开始吧！
