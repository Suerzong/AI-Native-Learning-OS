# Deep RL Course

## 简介

在上一单元中，我们学习了深度 Q-Learning。在这个基于价值的深度强化学习算法中，我们**使用深度神经网络来近似一个状态下每个可能动作的不同 Q 值**。

在课程开始以来，我们只研究了基于价值的方法，**在这些方法中我们估计价值函数作为寻找最优策略的中间步骤**。

在基于价值的方法中，策略**仅因为动作价值估计而存在**，因为策略只是一个函数（例如贪心策略），它将在给定状态下选择价值最高的动作。

使用基于策略的方法（Policy-Based Methods），我们希望**直接优化策略，而不需要学习价值函数的中间步骤**。

今天，我们将学习**基于策略的方法**并研究这些方法的一个子集——**策略梯度（Policy Gradient）**。然后，我们将使用 PyTorch 从头实现第一个策略梯度算法——蒙特卡洛 **REINFORCE**。

然后你将能够迭代和改进此实现以用于更高级的环境。

让我们开始吧！
