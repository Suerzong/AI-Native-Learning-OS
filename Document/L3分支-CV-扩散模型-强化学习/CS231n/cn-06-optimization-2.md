### 引言

**动机。** 本节我们将培养对**反向传播**的直观理解，这是一种通过递归应用**链式法则**计算表达式梯度的方法。理解此过程及其微妙之处对理解和有效开发、设计和调试神经网络至关重要。

**问题陈述。** 本节研究的核心问题是：给定某个函数 \\(f(x)\\)，其中 \\(x\\) 是输入向量，我们希望计算 \\(f\\) 在 \\(x\\) 处的梯度（即 \\(\nabla f(x)\\) ）。

**动机。** 回顾我们对此问题感兴趣的主要原因是在神经网络的具体情况下，\\(f\\) 对应损失函数 \\(L\\)，输入 \\(x\\) 包括训练数据和神经网络权重。例如，损失可以是 SVM 损失函数，输入是训练数据 \\((x_i,y_i), i=1 \ldots N\\) 以及权重和偏置 \\(W,b\\)。

### 简单表达式和梯度解释

让我们从简单开始。考虑两个数字的简单乘法函数 \\(f(x,y) = x y\\)。简单的微积分推导：

\\[f(x,y) = x y \hspace{0.5in} \rightarrow \hspace{0.5in} \frac{\partial f}{\partial x} = y \hspace{0.5in} \frac{\partial f}{\partial y} = x\\]

**解释。** 导数告诉你函数在特定点附近无穷小区域内对该变量的变化率。

> 每个变量上的导数告诉你整个表达式对其值的敏感度。

梯度 \\(\nabla f\\) 是偏导数的向量，因此 \\(\nabla f = [\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}] = [y, x]\\)。

加法操作的导数：

\\[f(x,y) = x + y \hspace{0.5in} \rightarrow \hspace{0.5in} \frac{\partial f}{\partial x} = 1 \hspace{0.5in} \frac{\partial f}{\partial y} = 1\\]

max 操作的导数：

\\[f(x,y) = \max(x, y) \hspace{0.5in} \rightarrow \hspace{0.5in} \frac{\partial f}{\partial x} = \mathbb{1}(x >= y) \hspace{0.5in} \frac{\partial f}{\partial y} = \mathbb{1}(y >= x)\\]

### 包含链式法则的复合表达式

考虑更复杂的表达式 \\(f(x,y,z) = (x + y) z\\)。此表达式可以分解为两个：\\(q = x + y\\) 和 \\(f = q z\\)。**链式法则**告诉我们将这些梯度表达式「链接」在一起的正确方式是通过乘法。例如 \\(\frac{\partial f}{\partial x} = \frac{\partial f}{\partial q} \frac{\partial q}{\partial x} \\)。


    # 设置输入
    x = -2; y = 5; z = -4

    # 执行前向传播
    q = x + y # q 变为 3
    f = q * z # f 变为 -12

    # 反向传播（反向传播）按逆序执行：
    # 首先反向传播 f = q * z
    dfdz = q # df/dz = q，所以 z 的梯度变为 3
    dfdq = z # df/dq = z，所以 q 的梯度变为 -4
    dqdx = 1.0
    dqdy = 1.0
    # 然后反向传播 q = x + y
    dfdx = dfdq * dqdx  # 这里的乘法就是链式法则！
    dfdy = dfdq * dqdy


### 反向传播的直觉理解

反向传播是一个美妙的局部过程。电路图中的每个门接收一些输入并可以立即计算两件事：1. 其输出值，2. 其输出关于其输入的「局部」梯度。一旦前向传播结束，在反向传播期间，门将最终了解其输出值对整个电路最终输出的梯度。链式法则说门应该获取该梯度并将其乘入其对所有输入通常计算的每个梯度中。

> 链式法则带来的额外乘法（对每个输入）可以将单个相对无用的门转变为复杂电路（如整个神经网络）中的齿轮。

### 模块化：Sigmoid 示例

\\[f(w,x) = \frac{1}{1+e^{-(w_0x_0 + w_1x_1 + w_2)}}\\]

此表达式描述了一个 2 维神经元（输入 **x** 和权重 **w**），使用「sigmoid 激活」函数。sigmoid 函数的导数可以简化为：

\\[\sigma(x) = \frac{1}{1+e^{-x}} \\\\\\\ \rightarrow \hspace{0.3in} \frac{d\sigma(x)}{dx} = \left( 1 - \sigma(x) \right) \sigma(x)\\]

此神经元反向传播的代码：


    w = [2,-3,-3] # 假设一些随机权重和数据
    x = [-1, -2]

    # 前向传播
    dot = w[0]*x[0] + w[1]*x[1] + w[2]
    f = 1.0 / (1 + math.exp(-dot)) # sigmoid 函数

    # 通过神经元反向传播
    ddot = (1 - f) * f # dot 变量的梯度，使用 sigmoid 梯度推导
    dx = [w[0] * ddot, w[1] * ddot] # 反向传播到 x
    dw = [x[0] * ddot, x[1] * ddot, 1.0 * ddot] # 反向传播到 w


**实现提示：分阶段反向传播。** 如上所示，在实践中将前向传播分解为易于反向传播的阶段总是有帮助的。

### 实践中的反向传播：分阶段计算

用另一个例子说明。假设我们有以下形式的函数：

\\[f(x,y) = \frac{x + \sigma(y)}{\sigma(x) + (x+y)^2}\\}

此函数完全无用，不清楚你为什么要计算其梯度，但它是实践反向传播的好例子。重要的是强调，如果你试图对 \\(x\\) 或 \\(y\\) 执行微分，最终会得到非常大且复杂的表达式。然而，事实证明这样做完全不必要，因为我们不需要写下来评估梯度的显式函数。


    x = 3 # 示例值
    y = -4

    # 前向传播
    sigy = 1.0 / (1 + math.exp(-y)) # 分子中的 sigmoid   #(1)
    num = x + sigy # 分子                               #(2)
    sigx = 1.0 / (1 + math.exp(-x)) # 分母中的 sigmoid #(3)
    xpy = x + y                                              #(4)
    xpysqr = xpy**2                                          #(5)
    den = sigx + xpysqr # 分母                        #(6)
    invden = 1.0 / den                                       #(7)
    f = num * invden # 完成！                                 #(8)

    # 反向传播
    dnum = invden #(8)
    dinvden = num #(8)
    dden = (-1.0 / (den**2)) * dinvden #(7)
    dsigx = (1) * dden #(6)
    dxpysqr = (1) * dden #(6)
    dxpy = (2 * xpy) * dxpysqr #(5)
    dx = (1) * dxpy #(4)
    dy = (1) * dxpy #(4)
    dx += ((1 - sigx) * sigx) * dsigx #(3)
    dx += (1) * dnum #(2)
    dsigy = (1) * dnum #(2)
    dy += ((1 - sigy) * sigy) * dsigy #(1)


注意几点：

**缓存前向传播变量。** 要计算反向传播，拥有前向传播中使用的一些变量非常有帮助。

**梯度在分叉处累加。** 前向表达式多次涉及变量 **x,y**，因此当我们执行反向传播时，必须小心使用 `+=` 而不是 `=` 来累积这些变量的梯度。

### 反向流中的模式

在许多情况下，反向流动的梯度可以在直觉层面上解释。神经网络中最常用的三个门（加、乘、max）在反向传播期间的行为都有非常简单的解释：

**加法门**始终获取其输出的梯度并将其平均分配给所有输入，无论它们在前向传播期间的值如何。

**Max 门**路由梯度。与加法门不同，Max 门将梯度（不变地）分配给其输入中值最大的那个。

**乘法门**不那么容易解释。其局部梯度是输入值（但交换了位置），然后在链式法则中乘以其输出的梯度。

### 向量化操作的梯度

以上各节涉及单个变量，但所有概念都以直接的方式扩展到矩阵和向量操作。

**矩阵-矩阵乘法梯度。** 最棘手的操作可能是矩阵-矩阵乘法：


    # 前向传播
    W = np.random.randn(5, 10)
    X = np.random.randn(10, 3)
    D = W.dot(X)

    # 假设我们从上面得到了 D 的梯度
    dD = np.random.randn(*D.shape) # 与 D 形状相同
    dW = dD.dot(X.T) # .T 给出矩阵的转置
    dX = W.T.dot(dD)


「提示：使用维度分析！」你不需要记住 `dW` 和 `dX` 的表达式，因为它们可以基于维度轻松重新推导。

### 总结

  * 我们开发了梯度意味着什么、它们如何在电路中向后流动、以及它们如何传达电路的哪些部分应该增加或减少以及以多大力度使最终输出更高的直觉。
  * 我们讨论了「分阶段计算」对反向传播实际实现的重要性。你总是希望将函数分解为可以轻松推导局部梯度的模块，然后用链式法则将它们链接起来。
