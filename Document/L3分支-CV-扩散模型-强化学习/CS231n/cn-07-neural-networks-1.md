## 快速入门

可以不借助大脑类比来介绍神经网络。在线性分类一节中，我们使用公式 \\( s = W x \\) 计算图像的不同视觉类别得分，其中 \\(W\\) 是矩阵，\\(x\\) 是包含图像所有像素数据的输入列向量。在 CIFAR-10 中，\\(x\\) 是 [3072x1] 列向量，\\(W\\) 是 [10x3072] 矩阵，输出是 10 个类别得分的向量。

一个示例神经网络则计算 \\( s = W_2 \max(0, W_1 x) \\)。这里 \\(W_1\\) 可以是 [100x3072] 矩阵，将图像转换为 100 维中间向量。函数 \\(max(0,-) \\) 是逐元素应用的非线性。非线性的选择有多种（我们将在下面学习），但这是一个常见选择，简单地将所有低于零的激活阈值化为零。最后，矩阵 \\(W_2\\) 大小为 [10x100]，我们再次得到 10 个数字作为类别得分。注意非线性在计算上至关重要——如果去掉它，两个矩阵可以折叠为一个矩阵，预测的类别得分将再次是输入的线性函数。

三层神经网络类似地为 \\( s = W_3 \max(0, W_2 \max(0, W_1 x)) \\)，所有 \\(W_3, W_2, W_1\\) 都是要学习的参数。

## 建模单个神经元

### 生物动机与连接

大脑的基本计算单元是**神经元**。人类神经系统中大约有 860 亿个神经元，通过约 10^14 - 10^15 个**突触**连接。下图显示了生物神经元的卡通图（左）和常见数学模型（右）。

![](/assets/nn1/neuron.png) ![](/assets/nn1/neuron_model.jpeg)

生物神经元的卡通图（左）及其数学模型（右）。

单个神经元的前向传播代码示例：


    class Neuron(object):
      # ...
      def forward(self, inputs):
        """ 假设 inputs 和 weights 是 1-D numpy 数组，bias 是数字 """
        cell_body_sum = np.sum(inputs * self.weights) + self.bias
        firing_rate = 1.0 / (1.0 + math.exp(-cell_body_sum)) # sigmoid 激活函数
        return firing_rate


换句话说，每个神经元与输入执行点积，加上偏置，然后应用非线性（激活函数），本例中是 sigmoid \\(\sigma(x) = 1/(1+e^{-x})\\)。

### 单个神经元作为线性分类器

模型神经元的前向计算的数学形式可能看起来熟悉。与线性分类器类似，神经元有能力「喜欢」（激活接近 1）或「不喜欢」（激活接近 0）其输入空间的某些线性区域。

**二元 Softmax 分类器**。例如，我们可以将 \\(\sigma(\sum_iw_ix_i + b)\\) 解释为某个类别的概率 \\(P(y_i = 1 \mid x_i; w) \\)。

**二元 SVM 分类器**。或者，我们可以在神经元的输出上附加最大间隔合页损失，训练它成为二元支持向量机。

> 单个神经元可用于实现二元分类器（例如二元 Softmax 或二元 SVM 分类器）

### 常用激活函数

![](/assets/nn1/sigmoid.jpeg) ![](/assets/nn1/tanh.jpeg)

**左图：** Sigmoid 非线性将实数压缩到 [0,1] 范围。**右图：** tanh 非线性将实数压缩到 [-1,1] 范围。

**Sigmoid。** Sigmoid 非线性数学形式为 \\(\sigma(x) = 1 / (1 + e^{-x})\\)。它将实数「压缩」到 0 到 1 的范围。Sigmoid 有两个主要缺点：

  * 「Sigmoid 会饱和并杀死梯度」。当神经元激活在 0 或 1 的任一端饱和时，这些区域的梯度几乎为零。
  * 「Sigmoid 输出不是零中心的」。这不理想，因为后续层的神经元会接收到非零中心的数据。

**Tanh。** Tanh 非线性将实数压缩到 [-1, 1] 范围。与 sigmoid 神经元不同，其输出是零中心的。因此，「在实践中 tanh 非线性总是优于 sigmoid 非线性。」\\( \tanh(x) = 2 \sigma(2x) -1 \\)。

![](/assets/nn1/relu.jpeg) ![](/assets/nn1/alexplot.jpeg)

**左图：** 整流线性单元（ReLU）激活函数，x < 0 时为零，x > 0 时斜率为 1。**右图：** ReLU 相比 tanh 单元收敛速度提高 6 倍。

**ReLU。** 整流线性单元近年非常流行。它计算函数 \\(f(x) = \max(0, x)\\)。

  * (+) 大大加速随机梯度下降的收敛（比 sigmoid/tanh 快约 6 倍）。
  * (+) 只需将激活矩阵阈值化为零即可实现。
  * (-) ReLU 单元在训练中可能很脆弱并可能「死亡」。如果学习率设置过高，可能有多达 40% 的网络「死亡」。

**Leaky ReLU。** Leaky ReLU 是修复「死亡 ReLU」问题的一种尝试。当 x < 0 时，不是让函数为零，而是有一个小的正斜率（如 0.01）。

**Maxout。** Maxout 神经元计算 \\(\max(w_1^Tx+b_1, w_2^Tx + b_2)\\)，推广了 ReLU 及其 leaky 版本。

**总结：** 使用 ReLU 非线性，注意学习率，可能监控网络中「死亡」单元的比例。如果担心，试试 Leaky ReLU 或 Maxout。永远不要用 sigmoid。可以试 tanh，但期望它比 ReLU/Maxout 差。

## 神经网络架构

### 层级组织

**神经网络作为图中的神经元。** 神经网络被建模为无环图中连接的神经元集合。对于常规神经网络，最常见的层类型是**全连接层**，其中两个相邻层之间的神经元完全成对连接，但单层内的神经元不共享任何连接。

![](/assets/nn1/neural_net.jpeg) ![](/assets/nn1/neural_net2.jpeg)

**左图：** 2 层神经网络（一个 4 个神经元的隐藏层和一个 2 个神经元的输出层）。**右图：** 3 层神经网络，两个各 4 个神经元的隐藏层和一个输出层。

**命名约定。** 当我们说 N 层神经网络时，不计算输入层。因此，单层神经网络描述没有隐藏层的网络（输入直接映射到输出）。这些网络也被称为「人工神经网络」（ANN）或「多层感知器」（MLP）。

**输出层。** 输出层神经元通常没有激活函数（或可以认为具有线性恒等激活函数）。

**衡量神经网络大小。** 人们常用两个指标：神经元数量，或更常见的参数数量。

  * 第一个网络（左）有 4 + 2 = 6 个神经元，[3 x 4] + [4 x 2] = 20 个权重和 4 + 2 = 6 个偏置，总共 26 个可学习参数。
  * 第二个网络（右）有 4 + 4 + 1 = 9 个神经元，[3 x 4] + [4 x 4] + [4 x 1] = 32 个权重和 9 个偏置，总共 41 个可学习参数。

### 前向传播计算示例

三层神经网络的前向传播只需三次矩阵乘法，交织着激活函数的应用：


    # 3 层神经网络的前向传播：
    f = lambda x: 1.0/(1.0 + np.exp(-x)) # 激活函数（使用 sigmoid）
    x = np.random.randn(3, 1) # 三个数字的随机输入向量 (3x1)
    h1 = f(np.dot(W1, x) + b1) # 计算第一隐藏层激活 (4x1)
    h2 = f(np.dot(W2, h1) + b2) # 计算第二隐藏层激活 (4x1)
    out = np.dot(W3, h2) + b3 # 输出神经元 (1x1)


> 全连接层的前向传播对应于一次矩阵乘法，后跟偏置偏移和激活函数。

### 表征能力

具有至少一个隐藏层的神经网络是「通用近似器」。也就是说，给定任何连续函数 \\(f(x)\\) 和某个 \\(\epsilon > 0\\)，存在一个具有一层隐藏层的神经网络 \\(g(x)\\)，使得 \\( \forall x, \mid f(x) - g(x) \mid < \epsilon \\)。

如果一层隐藏层足以近似任何函数，为什么使用更多层？答案是两层神经网络是通用近似器虽然数学上很漂亮，但在实践中是一个相对弱且无用的陈述。深层网络（多隐藏层）可以比单隐藏层网络工作得更好是经验观察。

### 设置层数和它们的大小

随着神经网络中大小和层数的增加，网络的**容量**增加。更大的神经网络可以表示更复杂的函数。但这也更容易过拟合。

![](/assets/nn1/layer_sizes.jpeg)

更大的神经网络可以表示更复杂的函数。

基于以上讨论，如果数据不够复杂，似乎更小的网络更可取。但这是不正确的——我们将在后面讨论许多其他更好的防止过拟合的方法（如 L2 正则化、dropout、输入噪声）。

要点是不要因为害怕过拟合而使用更小的网络。相反，你应该使用计算预算允许的尽可能大的神经网络，并使用其他正则化技术来控制过拟合。

## 总结

  * 我们介绍了生物**神经元**的粗略模型。
  * 我们讨论了实践中使用的几种**激活函数**，ReLU 是最常见的选择。
  * 我们介绍了**神经网络**，其中神经元通过**全连接层**连接。
  * 我们看到这种分层架构使得基于矩阵乘法的高效评估成为可能。
  * 我们看到神经网络是**通用函数近似器**。
  * 我们讨论了更大的网络总是比更小的网络工作得更好，但必须用更强的正则化适当地处理其更高的模型容量。
