## 设置数据和模型

在上一节中，我们介绍了神经元模型以及将神经元排列成层的神经网络。在本节中，我们将讨论数据预处理、权重初始化和损失函数方面的额外设计选择。

### 数据预处理

数据矩阵 `X` 有三种常见的预处理形式（假设 `X` 大小为 `[N x D]`，`N` 是数据量，`D` 是维度）。

**均值减法**是最常见的预处理。它涉及减去数据中每个特征的均值，几何解释是将数据云沿每个维度居中到原点。在 numpy 中：`X -= np.mean(X, axis = 0)`。对于图像，常见做法是减去所有像素的单一值（例如 `X -= np.mean(X)`），或在三个颜色通道上分别减去。

**归一化**指的是将数据维度归一化到大致相同的尺度。一种方法是在零中心化后将每个维度除以其标准差：`X /= np.std(X, axis = 0)`。

**PCA 和白化**是另一种预处理形式。数据首先被中心化，然后计算协方差矩阵，通过 SVD 分解实现去相关和降维。

在实践中，零中心化数据非常重要，对每个像素的归一化也很常见。

### 权重初始化

**陷阱：全零初始化。** 如果每个神经元的权重初始化为相同值，它们将计算相同的输出和梯度，进行完全相同的参数更新。没有不对称性来源。

**小随机数。** 因此，将权重初始化为接近零但不完全相同的随机小数。`W = 0.01* np.random.randn(D,H)`。

**用 1/sqrt(n) 校准方差。** 随机初始化的神经元输出方差随输入数量增长。通过将其权重向量乘以其「扇入」（fan-in）的平方根的倒数来归一化方差：`w = np.random.randn(n) / sqrt(n)`。

Glorot 等人的论文推荐 \\( \text{Var}(w) = 2/(n_{in} + n_{out}) \\)。He 等人的论文为 ReLU 神经元推荐 `w = np.random.randn(n) * sqrt(2.0/n)`。

**初始化偏置。** 通常将偏置初始化为零。

**批归一化。** Ioffe 和 Szegedy 开发的[批归一化](http://arxiv.org/abs/1502.03167)通过显式强制整个网络的激活在训练开始时服从单位高斯分布，缓解了很多初始化方面的麻烦。在实践中，使用批归一化的网络对不良初始化明显更鲁棒。

### 正则化

有几种控制神经网络容量以防止过拟合的方法：

**L2 正则化**可能是最常见的正则化形式。通过对所有参数的平方幅度直接在目标中惩罚实现：\\(\frac{1}{2} \lambda w^2\\)。在梯度下降更新中，使用权重衰减：`W += -lambda * W`。

**L1 正则化**：对每个权重 \\(w\\) 添加 \\(\lambda \mid w \mid\\) 到目标。L1 正则化导致权重向量在优化过程中变得稀疏（即非常接近零）。

**Max norm 约束。** 强制每个神经元权重向量的绝对上限。

**Dropout** 是一种极其有效的正则化技术。在训练期间，以概率 \\(p\\) 保持神经元激活，否则将其设为零。

![](/assets/nn2/dropout.jpeg)

Vanilla dropout 实现：


    """ Vanilla Dropout: 不推荐的实现 """
    p = 0.5 # 保持单元激活的概率

    def train_step(X):
      H1 = np.maximum(0, np.dot(W1, X) + b1)
      U1 = np.random.rand(*H1.shape) < p
      H1 *= U1
      H2 = np.maximum(0, np.dot(W2, H1) + b2)
      U2 = np.random.rand(*H2.shape) < p
      H2 *= U2
      out = np.dot(W3, H2) + b3

    def predict(X):
      H1 = np.maximum(0, np.dot(W1, X) + b1) * p
      H2 = np.maximum(0, np.dot(W2, H1) + b2) * p
      out = np.dot(W3, H2) + b3


**反向 Dropout**（推荐实现）在训练时进行缩放，测试时不做任何操作：


    """ 反向 Dropout: 推荐实现 """
    p = 0.5

    def train_step(X):
      H1 = np.maximum(0, np.dot(W1, X) + b1)
      U1 = (np.random.rand(*H1.shape) < p) / p
      H1 *= U1
      H2 = np.maximum(0, np.dot(W2, H1) + b2)
      U2 = (np.random.rand(*H2.shape) < p) / p
      H2 *= U2
      out = np.dot(W3, H2) + b3

    def predict(X):
      H1 = np.maximum(0, np.dot(W1, X) + b1)
      H2 = np.maximum(0, np.dot(W2, H1) + b2)
      out = np.dot(W3, H2) + b3


在实践中：最常见的是使用单一的全局 L2 正则化强度（通过交叉验证）。通常也与所有层后的 dropout 结合使用。\\(p = 0.5\\) 是合理的默认值。

### 损失函数

目标的第二部分是「数据损失」，在监督学习问题中衡量预测与真实标签之间的兼容性。

**分类**：SVM（Weston Watkins 公式）\\(L_i = \sum_{j\neq y_i} \max(0, f_j - f_{y_i} + 1)\\)，或 Softmax 交叉熵损失 \\(L_i = -\log\left(\frac{e^{f_{y_i}}}{ \sum_j e^{f_j} }\right)\\)。

**属性分类**：当标签不是互斥时，对每个属性独立构建二元分类器。

**回归**：预测实值量。L2 损失 \\(L_i = \Vert f - y_i \Vert_2^2\\)，L1 损失 \\(L_i = \Vert f - y_i \Vert_1\\)。

> 面对回归任务时，首先考虑是否绝对必要。尽可能将输出离散化为区间并对其执行分类。

## 总结

  * 推荐的预处理是将数据中心化为零均值，并沿每个特征将其尺度归一化到 [-1, 1]
  * 用标准差 \\(\sqrt{2/n}\\) 的高斯分布初始化权重，其中 \\(n\\) 是神经元的输入数
  * 使用 L2 正则化和 dropout（反向版本）
  * 使用批归一化
  * 我们讨论了实践中可能执行的不同任务及其最常见的损失函数
