# Env

Gymnasium 的主要类，用于实现强化学习智能体环境。

该类通过 `step()` 和 `reset()` 函数封装了具有任意后台动态的环境。环境可以被单个智能体部分或完全观测。

## 主要 API 方法

  * `step()` —— 使用动作更新环境，返回下一个观测、奖励、是否终止或截断以及环境信息。
  * `reset()` —— 将环境重置为初始状态，返回第一个观测和信息。
  * `render()` —— 渲染环境以帮助可视化智能体所见。
  * `close()` —— 关闭环境，清理资源。

## 属性

  * `action_space` —— 对应有效动作的空间对象。
  * `observation_space` —— 对应有效观测的空间对象。
  * `metadata` —— 环境的元数据，如渲染模式和帧率。
  * `spec` —— 环境规格，包含从 `make()` 初始化环境的信息。
  * `np_random` —— 环境的随机数生成器。

## 方法详情

### step(action)

使用智能体动作运行一个时间步。返回：
- **observation**：下一个观测
- **reward**：执行动作后的奖励
- **terminated**：智能体是否达到终止状态
- **truncated**：是否因时间限制等原因被截断
- **info**：辅助诊断信息

### reset(seed=None, options=None)

将环境重置为初始内部状态，返回初始观测和信息。

### render()

根据初始化时指定的 `render_mode` 计算渲染帧。

### close()

在用户使用完环境后，执行必要的「清理」工作。
