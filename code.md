简短教学：池化层（Pooling）
什么是池化
池化 = 缩小特征图，保留重要信息。

最常用的是 Max Pooling（最大池化）：在每个小区域里取最大值。


输入（4×4）：           Max Pooling（2×2，stride=2）：



┌──────────────┐       ┌─────────┐
│ 1  3 │ 2  4  │       │ 3 │ 4   │
│ 5  2 │ 1  3  │  →    ├───┼─────│
│──────┼───────│       │ 6 │ 8   │
│ 4  6 │ 7  8  │       └─────────┘
│ 2  1 │ 3  5  │
└──────────────┘       输出（2×2）



每个 2×2 区域取最大值：
左上：max(1,3,5,2) = 3
右上：max(2,4,1,3) = 4
左下：max(4,6,2,1) = 6
右下：max(7,8,3,5) = 8
池化的作用
作用	说明
缩小尺寸	4×4 → 2×2，减少计算量
保留重要特征	取最大值 = 保留最强的特征信号
平移不变性	特征稍微移动位置，池化结果不变
池化没有参数
池化层没有可学习的权重，只是固定操作（取最大或取平均）。
