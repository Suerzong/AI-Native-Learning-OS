# 空课程：Edge AI 综合项目 课程配置

本文件是课程模板。当前课程尚未填充正式教学内容，不能据此记录为已学习能力。

---

# 课程身份

- 课程名：Edge AI 综合项目
- 当前状态：空课程
- 目标：完成从需求定义、数据采集、模型训练、端侧部署到系统演示的完整 Edge AI 工程闭环
- 前置知识：C/C++、Python、MCU/嵌入式 Linux、OpenCV、PyTorch、模型部署基础
- 对应能力模块：模块 12（工程项目、机器人系统与科研能力）为主，联动模块 5-11
- 课程规模：待定，建议以 4-6 个阶段项目组织
- 课程定位：把分散课程能力整合成可展示、可复现、可迭代的工程作品

---

# 课程架构模板

```
第一层：工程准备
  需求定义 -> 技术选型 -> 项目结构 -> Git/文档 -> 评估指标

第二层：数据与模型
  数据采集 -> 数据清洗/标注 -> 训练 -> 评估 -> 实验记录

第三层：端侧部署
  模型导出 -> 推理验证 -> 预处理一致性 -> 性能 benchmark

第四层：系统集成与展示
  传感器/摄像头 -> 推理 -> 通信/控制 -> 稳定性测试 -> 报告与演示
```
