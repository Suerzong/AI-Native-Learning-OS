# 每日诊断短测

每日短测用于判断今天应该训练什么，而不是为了打分。

## 模板

```md
## YYYY-MM-DD

### 诊断题

1. 概念题：
2. 工具题：
3. 流程题：
4. 调试题：

### 结果

- 正确率：
- 主要错误：
- 今日训练目标：
- 是否允许推进：
```

---

## 文本预处理诊断样例

1. 中文文本预处理的基本流程是什么？按顺序列出各个步骤。
2. jieba 的精确模式、全模式、搜索引擎模式有什么区别？
3. 给定句子"我来到北京邮电大学"，写出 jieba 精确模式的分词结果。
4. 如果分词后出现乱码，最可能的原因是什么？怎么解决？

## 文本表示诊断样例

1. 词袋模型（BoW）的两个主要局限性是什么？
2. TF-IDF 中 TF 和 IDF 分别衡量什么？写出公式。
3. 词嵌入（Word2Vec）和 one-hot 编码的核心区别是什么？
4. 给定 3 篇文档，某词在所有文档中都出现，它的 IDF 值会怎样？

## 文本分类诊断样例

1. 朴素贝叶斯分类器的"朴素"体现在哪里？
2. 文本分类的标准 pipeline 包括哪几步？
3. 为什么 TF-IDF + 朴素贝叶斯是一个常用的文本分类 baseline？
4. sklearn 中 `fit_transform` 和先 `fit` 再 `transform` 的区别是什么？

## NER 诊断样例

1. NER 的任务目标是什么？举出 3 种常见实体类型。
2. BIO 标注方案中 B、I、O 分别代表什么？
3. 对句子"李明在上海交通大学读书"进行 BIO 标注（人名+机构名）。
4. 中文 NER 相比英文 NER 的主要难点是什么？

## RNN 与序列模型诊断样例

1. RNN 相比普通神经网络处理序列数据的核心优势是什么？
2. RNN 的梯度消失问题会导致什么后果？
3. LSTM 相比普通 RNN 增加了什么机制来解决长距离依赖？
4. 在 PyTorch 中，`nn.LSTM` 的 output 和 (h_n, c_n) 分别是什么？

## 注意力机制与 Transformer 诊断样例

1. 注意力机制中 Q、K、V 分别代表什么？
2. Scaled Dot-Product Attention 的公式是什么？为什么要除以 sqrt(d_k)？
3. Multi-Head Attention 的"多头"有什么作用？
4. Transformer 相比 RNN 的两个核心优势是什么？

## BERT 与 GPT 诊断样例

1. BERT 的两个预训练目标是什么？
2. BERT 和 GPT 在架构上的核心区别是什么？
3. Hugging Face Tokenizer 输出的 input_ids、attention_mask 分别是什么含义？
4. 微调 BERT 时学习率通常设置为多少？为什么不能太大？

## 端侧部署诊断样例

1. NLP 模型部署到嵌入式端面临的主要挑战有哪些？
2. BERT-base 模型有多少参数？在 MCU 上能直接运行吗？
3. 模型量化能减小模型大小，但可能带来什么副作用？
4. 知识蒸馏的核心思想是什么？
