# 文本预处理 技能地图

## 目标

学习者能完成从原始文本到干净词序列的完整预处理流程，包括清洗、分词、去停用词、标准化。

## 必会概念

- 文本清洗（去除 HTML 标签、特殊字符、统一编码）
- 分词（中文分词 vs 英文分词）
- 停用词（高频无意义词：的、了、是、the、a、is）
- 标准化（统一大小写、词干化/词形还原）
- 编码处理（UTF-8、GBK）

## 关键流程

```
原始文本
  ↓
编码处理（确保 UTF-8）
  ↓
文本清洗（去 HTML、特殊字符、多余空白）
  ↓
分词（jieba / nltk / spaCy）
  ↓
去停用词
  ↓
标准化（小写化 / 词干化）
  ↓
干净的词列表
```

## 必会工具

| 工具 | 用途 | 安装 |
|------|------|------|
| `jieba` | 中文分词 | `pip install jieba` |
| `re` (Python 内置) | 正则表达式清洗 | 无需安装 |
| `chardet` | 编码检测 | `pip install chardet` |

## 代码示例

```python
import jieba
import re

# 1. 原始文本
raw_text = "<p>北京邮电大学是中国信息科技领域的顶尖高校！！！</p>"

# 2. 文本清洗
text = re.sub(r'<[^>]+>', '', raw_text)  # 去 HTML 标签
text = re.sub(r'[^一-龥a-zA-Z0-9]', '', text)  # 只保留中英文和数字
text = text.strip()

# 3. 中文分词
words = jieba.lcut(text)
print("分词结果:", words)
# ['北京邮电大学', '是', '中国', '信息', '科技', '领域', '的', '顶尖', '高校']

# 4. 去停用词
stopwords = set(['的', '了', '是', '在', '和', '有', '也', '都', '就'])
filtered = [w for w in words if w not in stopwords and len(w) > 0]
print("去停用词:", filtered)
# ['北京邮电大学', '中国', '信息', '科技', '领域', '顶尖', '高校']
```

## 常见错误

1. **中文不分词直接处理**：中文没有空格，必须用分词工具
2. **编码乱码**：文件编码和读取编码不一致（GBK vs UTF-8）
3. **停用词表不完整**：不同领域需要不同的停用词表
4. **过度清洗**：去掉标点后丢失了问号、感叹号等情感信息
5. **jieba 未加载词典**：专业术语被错误切分

## 训练阶梯

1. **基础清洗**：能用正则表达式去除 HTML 标签和特殊字符
2. **中文分词**：能用 jieba 做精确分词
3. **去停用词**：能加载和使用停用词表
4. **完整流程**：能写出从原始文本到干净词列表的完整 pipeline
5. **进阶**：能处理编码问题、自定义词典、不同分词模式

## 掌握标准

- 能写出完整的文本预处理 pipeline
- 能处理中文编码问题
- 能对比不同分词模式的结果
- 能自定义停用词表和 jieba 用户词典
- 能解释每一步的目的和可能丢失的信息
