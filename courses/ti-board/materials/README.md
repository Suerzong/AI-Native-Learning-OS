# 资料目录

本目录存放训练 agent 需要引用的官方资料、SDK 示例、数据手册和索引文件。

## 当前板卡

- 开发板：LP-MSPM0G3507
- 芯片：MSPM0G3507
- 训练方向：嵌入式基础、外设控制、系统实验、Edge AI 示例

## 目录说明

| 路径 | 用途 |
|---|---|
| `raw-sdk/` | 精简后的 TI MSPM0 SDK |
| `datasheets/` | 板卡指南、子系统设计文档、应用笔记 |
| `sdk-index.md` | SDK 根目录和关键子目录说明 |
| `examples-map.md` | 本地示例到技能点的映射 |
| `board-info.md` | 开发板资源摘要 |
| `chip-info.md` | 芯片能力摘要 |
| `sysconfig-notes.md` | SysConfig 学习笔记 |
| `competition-focus.md` | 面向电赛/项目的训练重点 |
| `references.md` | 资料清单和个性化补充记录 |

## 个性化修改

- 如果新增 PDF，把文件名和用途写入 `references.md`。
- 如果新增 SDK 示例索引，把路径写入 `examples-map.md`。
- 如果发现板卡引脚或跳帽信息，整理到 `board-info.md`。
