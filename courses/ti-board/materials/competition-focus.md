# 面向比赛/项目的训练重点

如果学习目标包含电赛、创新项目或快速工程实现，训练重点应从“会一个外设”推进到“能组合系统并调试”。

## 优先技能

| 优先级 | 技能 | 原因 |
|---:|---|---|
| 1 | GPIO / 中断 | 人机交互、状态输入、事件响应 |
| 2 | Timer / PWM | 控制输出、周期任务 |
| 3 | ADC | 传感器采样 |
| 4 | UART | 调试输出、模块通信 |
| 5 | I2C / SPI | 常见传感器和外设 |
| 6 | DMA | 高频采样和数据搬运 |
| 7 | Edge AI | 分类、识别、异常检测 |

## 项目化训练路线

1. 按键控制 LED
2. PWM 控制 LED 亮度
3. ADC 采样并串口输出
4. ADC + 滤波
5. 传感器数据采集
6. 简单分类或阈值判断
7. Edge AI 示例迁移

## 个性化修改

如果有具体比赛题目，把题目拆成：

- 输入是什么
- 输出是什么
- 需要哪些外设
- 实时性要求
- 精度要求
- 最小可运行版本
