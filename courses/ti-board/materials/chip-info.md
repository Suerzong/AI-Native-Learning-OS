# MSPM0G3507 芯片信息

## 基本定位

MSPM0G3507 是 TI MSPM0 系列 Arm Cortex-M0+ MCU，适合低功耗控制、外设通信、传感器采集和轻量级边缘 AI 示例。

## 学习时重点关注

| 能力 | 学习价值 |
|---|---|
| GPIO | 所有外设和板卡实验的入口 |
| Timer / PWM | 控制周期、频率、占空比 |
| ADC | 传感器模拟采样 |
| UART / I2C / SPI | 外设通信 |
| DMA | 高频数据搬运 |
| Flash | 参数保存、非易失存储 |
| 中断 | 响应外部事件和周期事件 |

## Agent 使用方式

- 讲外设时先用 `skill-map/` 的技能地图。
- 查具体 API 时看 `source/ti/driverlib`。
- 查寄存器和硬件限制时补充技术参考手册或数据手册。
