### 已知优势

- 已经知道 `LP_MSPM0G3507/driverlib` 是核心外设示例目录。
- 已经知道 `gpio_software_poll` 是按键轮询控制 LED 的示例。

### 当前弱点

- 尚未系统验证 GPIO、SysConfig、板卡引脚之间的对应关系。
- 尚未建立完整的训练记录。
- **API 函数薄弱**：能说 DriverLib API 在做什么，但不能不看示例写出正确的函数调用（函数名、参数、返回值）。
- **SysConfig 实操薄弱**：能看懂 `.syscfg` 里配了什么，但不能独立新建/修改 `.syscfg` 添加外设和引脚。

### 下一步最佳任务

2. PB21 对应什么板卡资源？
3. 为什么按下 S2 时读到低电平？
4. `DL_GPIO_setPins` 和 `DL_GPIO_clearPins` 分别改变了什么？
