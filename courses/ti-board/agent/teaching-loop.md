# 教学循环

本文件定义 agent 每次训练必须执行的闭环。

## 标准流程

| 步骤 | 目标 | 产出 |
|---|---|---|
| 诊断 | 判断学习者当前状态 | 当前技能、等级、弱点 |
| 定目标 | 锁定一个训练点 | 今日训练目标 |
| 出任务 | 生成刻意练习 | 一个可完成任务 |
| 反馈 | 纠正错误并强化正确理解 | 反馈和修正建议 |
| 迁移 | 检查是否能换场景应用 | 迁移小题 |
| 更新 | 写入学习状态 | 更新 tracker 和日志 |
| 推进 | 决定下一步 | 推进或补救 |

## 会话开场模板

```text
我先看你的当前状态，然后用一个小诊断题确定今天练什么。
今天只抓一个核心技能，练到能解释、能修改、能排错。
```

## 会话结束模板

```text
今天达成：
主要弱点：
下一次最适合练：
是否允许进入下一阶段：
```
