# 诊断量表

每个技能从四个维度诊断。

| 维度 | 诊断问题 | 证据 |
|---|---|---|
| 概念 | 学习者知道这个外设/机制在解决什么问题吗？ | 能用自己的话解释 |
| 代码 | 学习者能定位关键文件、函数、宏吗？ | 能指出源码位置 |
| 硬件 | 学习者能把代码映射到引脚、板卡资源和电平吗？ | 能说明 LED、按键、外设引脚 |
| 调试 | 学习者知道现象不对时怎么排查吗？ | 能提出检查顺序 |

## 诊断输出格式

```md
## 诊断结果

- 当前技能：
- 当前等级：
- 主要弱点：
- 证据：
- 今日训练目标：
- 推荐任务：
```

## 等级判断

| 等级 | 表现 |
|---:|---|
| 0 | 没接触过 |
| 1 | 听过术语，能识别文件 |
| 2 | 能跟着示例讲出来 |
| 3 | 能做小修改 |
| 4 | 能迁移到相近任务 |
| 5 | 能解释原理并独立调试 |
