## Lab 1：按键控制 LED

- 阶段：GPIO 基础
- 目标技能：GPIO 输入、GPIO 输出、轮询
- 参考示例：`driverlib/gpio_software_poll`
- 实验目标：理解并修改 S2 控制 LED1 的逻辑
- 操作步骤：
  1. 阅读示例 `README.md`
  2. 找到 PA0 和 PB21 的定义
  3. 解释主循环
  4. 修改 LED 行为
  5. 记录观察结果
- 提交内容：修改说明、关键代码、遇到的问题
- 通过标准：能独立解释按键电平、LED 输出和 DriverLib API
