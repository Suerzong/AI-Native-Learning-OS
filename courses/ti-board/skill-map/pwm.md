# PWM 技能地图

## 目标

学习者能使用定时器/PWM 控制 LED 亮度或输出周期信号。

## 必会概念

- 周期
- 频率
- 占空比
- 定时器
- 输出比较

## 本地示例

| 示例 | 用途 |
|---|---|
| `driverlib/timer_*` | 定时器基础 |
| `msp_subsystems/pushbutton_change_pwm` | 按键改变 PWM |
| `msp_subsystems/pwm_led_driver` | PWM LED 控制 |

## 掌握标准

- 能解释频率和占空比。
- 能修改占空比改变输出行为。
- 能说明 PWM 和普通 GPIO 输出的区别。
