# Timer 技能地图

## 目标

学习者能理解定时器如何产生周期事件，并能把定时器用于延时、周期任务和 PWM 基础。

## 必会概念

- 计数器
- 时钟源
- 周期
- 频率
- 定时中断
- 捕获/比较

## 本地示例

| 示例 | 用途 |
|---|---|
| `driverlib/timer_*` | 定时器基础 |
| `driverlib/systick_*` | 系统滴答 |
| `msp_subsystems/task_scheduler` | 周期任务调度 |

## 掌握标准

- 能解释定时器为什么需要时钟源。
- 能把周期和频率互相换算。
- 能找到定时器中断入口。
- 能说明定时器和 `while` 软件延时的区别。
