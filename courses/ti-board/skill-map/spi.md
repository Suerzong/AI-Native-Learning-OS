# SPI 技能地图

## 目标

学习者能理解 SPI 主从通信和基础收发流程。

## 必会概念

- SCLK
- MOSI / MISO
- CS
- 时钟极性和相位
- 全双工通信

## 本地示例

| 示例 | 用途 |
|---|---|
| `driverlib/spi_*` | SPI 基础示例 |
| `msp_subsystems/uart_to_spi_bridge` | UART 到 SPI 桥 |
| `msp_subsystems/spi_to_i2c_bridge` | SPI 到 I2C 桥 |

## 掌握标准

- 能画出 SPI 四根常见信号线。
- 能解释 CS 的作用。
- 能说明 SPI 和 I2C 的基本区别。
