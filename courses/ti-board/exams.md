# 阶段考核

阶段考核用于判断是否形成稳定能力，而不是检查是否背过答案。

## 考核模板

```md
## 阶段名称

- 覆盖技能：
- 考核时间：
- 允许资料：
- 任务：
- 评分标准：
- 晋级条件：
```

## GPIO 阶段考核

- 覆盖技能：GPIO 输出、GPIO 输入、上拉、低有效、轮询
- 考核时间：30 分钟
- 允许资料：SDK 示例、板卡用户指南、自己的笔记
- 任务：
  1. 解释 `gpio_software_poll` 的硬件和代码关系
  2. 修改 LED 行为
  3. 设计一个“长按和短按表现不同”的伪代码方案
  4. 说明如果 LED 不亮，你会按什么顺序排查
- 晋级条件：
  - 正确率不低于 85%
  - 能独立定位关键文件
  - 能解释至少一个调试路径
