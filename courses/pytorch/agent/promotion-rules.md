# 晋级规则

只有出现稳定证据时才允许进入下一阶段。

## 通用晋级条件

学习者必须同时满足：

- 能用自己的话解释核心概念。
- 能不查文档写出核心 API 的正确调用（函数名、参数、返回值）。
- 能预测操作后的 tensor shape。
- 能修改代码并解释改动的影响。
- 能完成一个近迁移任务（换数据集或换参数）。
- 最近测试正确率不低于 80%。
- 没有未纠正的关键误区。

## 阶段性证据

| 等级 | 晋级证据 |
|---:|---|
| 1 -> 2 | 能识别术语和代码，能跟着示例运行 |
| 2 -> 3 | 能跟做并解释每一步的 shape 变化 |
| 3 -> 4 | 能独立修改代码并完成迁移任务 |
| 4 -> 5 | 能从零写出完整代码并调试错误 |

## 不允许推进的情况

- 学习者只能复述答案，不能解释为什么。
- 学习者能写代码但说不出每一步的 tensor shape。
- 同一误区连续出现两次。
- 没有做迁移测试。

## 层级晋级条件

### 第一层 → 第二层

必须通过：
- 能创建各种形状和类型的 tensor
- 能预测 tensor 运算后的 shape
- 能使用 reshape/squeeze/unsqueeze/permute
- 能解释 requires_grad 和 backward 的机制
- 能解释计算图的概念
- 考核正确率不低于 80%

### 第二层 → 第三层

必须通过：
- 能从零定义 nn.Module 子类（含 __init__ 和 forward）
- 能计算 Conv2d、MaxPool2d 等层的输出 shape
- 能写出完整训练循环（含 train/eval 切换）
- 能创建自定义 Dataset 和 DataLoader
- MNIST 准确率不低于 95%
- 考核正确率不低于 80%

### 第三层 → 第四层

必须通过：
- CIFAR-10 准确率不低于 75%（简单 CNN）
- 迁移学习准确率不低于 85%（预训练 ResNet18）
- 能计算 precision/recall/F1
- 能使用 TensorBoard 记录训练过程
- 能保存和加载 state_dict
- 能完成自定义数据集项目
- 考核正确率不低于 80%

### 第四层（结业）

必须通过：
- 能将模型导出为 ONNX 并验证推理一致性
- 能使用 TorchScript 导出
- 能做模型量化并分析精度/大小变化
- 能画出完整的端侧部署流程图
- 考核正确率不低于 80%
