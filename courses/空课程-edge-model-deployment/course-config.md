# 空课程：模型部署与端侧优化 课程配置

本文件是课程模板。当前课程尚未填充正式教学内容，不能据此记录为已学习能力。

# 课程身份

- 课程名：模型部署与端侧优化
- 当前状态：空课程
- 目标：掌握训练模型到端侧推理的完整流程，包括导出、转换、验证、量化、性能分析和平台适配
- 前置知识：Python、PyTorch 或 TensorFlow、嵌入式 Linux 基础、C/C++ 基础
- 对应能力模块：模块 11（模型部署、端侧优化与 AI 硬件平台）、模块 10（机器学习与深度学习）、模块 8（嵌入式 Linux 系统）、模块 12（工程项目与科研能力）
- 课程规模：待定，建议 42 节
- 课程定位：把训练好的模型变小、变快、变稳定，并让它在端侧平台真正运行

---

# 课程架构模板

```
第一层：模型导出与验证
  state_dict -> ONNX -> TorchScript -> SavedModel -> TFLite

第二层：推理运行时
  ONNX Runtime -> TFLite Interpreter -> OpenVINO/NCNN/MNN 概念 -> C++ API

第三层：端侧优化
  量化 -> 剪枝 -> 蒸馏概念 -> 算子支持 -> 性能指标

第四层：平台项目
  PC 验证 -> 嵌入式 Linux -> Jetson/TensorRT -> RKNN/RK3588 -> TinyML 概览
```
