# 工科数学分析

本目录用于工科数学分析 / 高等数学类课程的纯数学教学。

## 课程边界

- 只教学数学内容：定义、定理、证明思路、例题、习题、错题、复测和考试训练。
- 不引入 AI、编程、信号处理、电路、嵌入式或工程项目内容。
- 教学顺序严格以 `materials/toc.md` 为唯一依据。
- 教辅作为训练主材料，教材作为概念、定义、定理和证明来源。

## 使用入口

- 掌握学习：`/mastery-learn engineering-math-analysis`
- 别名：`math-analysis`、`advanced-math`、`高等数学`、`工科数学分析`
- 影印照片入口：`materials/inbox/scans/<batch-id>/`

## 文件说明

| 文件 | 用途 |
|---|---|
| `course-config.md` | 课程身份、教学边界、知识图谱和诊断规则 |
| `mastery-tracker.md` | 按目录顺序跟踪掌握度 |
| `mistakes.md` | 数学错题和误区记录 |
| `daily-tests.md` | 每日短测记录 |
| `exercises.md` | 练习与变式题沉淀 |

| `learner/state.md` | 当前学习位置和薄弱点 |

| `materials/source-manifest.md` | 影印资料批次与来源记录 |
| `materials/typing-conventions.md` | 键盘输入数学公式约定 |
