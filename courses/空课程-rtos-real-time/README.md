# RTOS 与实时系统

本课程面向 Edge AI 嵌入式设备的实时操作系统工程教学，从裸机思维过渡到多任务实时系统设计。

- 状态：课程配置完成，待首次教学
- 主教材：《Mastering the FreeRTOS Real-Time Kernel》（Richard Barry, v1.1.0），13 章
- 源码参考：FreeRTOSv202604.00-LTS 内核源码
- 课程规模：40 节（教材 37 节 + 综合项目 3 节）
- 配置日期：2026-05-22

对应能力模块：
- 7. RTOS 与实时系统
- 5. 单片机与裸机开发
- 6. 传感器、通信与外设接口
- 12. 工程项目、机器人系统与科研能力

课程架构：
1. Ch1-Ch3：RTOS 基础与内核配置（概念→构建→内存管理）
2. Ch4：任务管理（任务状态/优先级/调度/Tick/延时）——最核心章节
3. Ch5-Ch6：IPC 入门（队列 + 软件定时器）
4. Ch7-Ch8：中断与资源管理（ISR 协同/信号量/Mutex/优先级反转）
5. Ch9-Ch11：高级 IPC 与低功耗（事件组/任务通知/Tickless）
6. Ch12-Ch13：开发支持与调试（栈检测/运行时统计/故障排查）
7. 综合项目：多传感器采集→控制命令→Edge AI 推理接口
