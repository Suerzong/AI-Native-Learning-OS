# 练习库

所有练习按 FreeRTOS 章节编号。格式：`Ch{章}-{类型}-{序号}`

类型：CONCEPT（概念题）、API（API 使用题）、DESIGN（架构设计题）、DEBUG（故障排查题）、COMPARE（对比选择题）

出题前必须先判断任务是否属于低价值题。只考死记 API 参数顺序的题不应进入本任务库。练习重点放在概念辨析、IPC 选择、调度时序分析和故障排查。

## 任务模板

```md
## 任务名称

- 价值判断：这道题为什么不是低价值机械题？
- 目标技能：
- 起始材料：
- 任务要求：
- 限制条件：
- 成功标准：
- 常见错误：
- 反馈重点：
```
