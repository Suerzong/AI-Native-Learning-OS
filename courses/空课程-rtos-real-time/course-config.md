# 课程身份

- 课程名：RTOS 与实时系统
- 当前状态：空课程
- 目标：掌握 FreeRTOS / RT-Thread 的任务调度、同步通信、实时性分析和嵌入式多任务系统设计
- 前置知识：C 语言、STM32/MCU 裸机开发、中断、定时器、基础数据结构
- 对应能力模块：模块 7（RTOS 与实时系统）、模块 5（单片机与裸机开发）、模块 6（传感器、通信与外设接口）、模块 12（工程项目与科研能力）
- 课程规模：待定，建议 32-42 节
- 课程定位：从裸机程序过渡到复杂嵌入式实时系统，为 Edge AI 设备的数据采集、推理调度和控制执行打基础

---

# 课程架构模板

```
第一层：RTOS 基础概念
  实时系统概念 -> 任务/线程 -> 调度与优先级 -> Tick 与延时

第二层：任务通信与同步
  队列 -> 信号量 -> 互斥锁 -> 事件组 -> 软件定时器

第三层：中断协同与资源管理
  ISR 与任务通信 -> 栈分析 -> 内存管理 -> 优先级反转 -> 死锁排查

第四层：综合项目
  多任务传感器采集 -> 通信上报 -> 控制执行 -> 稳定性测试
```
