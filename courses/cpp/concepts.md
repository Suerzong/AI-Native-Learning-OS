## C vs C++ 对照

| C 写法 | C++ 写法 | 说明 |
|--------|---------|------|
| `printf("Hello")` | `std::cout << "Hello"` | 输出 |
| `scanf("%d", &n)` | `std::cin >> n` | 输入 |
| `malloc(sizeof(int))` | `new int` | 动态分配 |
| `free(p)` | `delete p` | 释放 |
| `char s[100]` | `std::string s` | 字符串 |
| `strcat(s1, s2)` | `s1 += s2` | 字符串拼接 |
| `void swap(int* a, int* b)` | `void swap(int& a, int& b)` | 交换 |
| `typedef struct { ... } Foo;` | `class Foo { ... };` | 类型定义 |
| `init_xxx()` / `destroy_xxx()` | 构造函数 / 析构函数 | 生命周期 |
| 函数指针表模拟多态 | virtual 虚函数 | 多态 |
| `qsort(arr, n, sizeof(int), cmp)` | `std::sort(v.begin(), v.end())` | 排序 |
| 手写链表 | `std::list` | 链表 |
| 手写哈希表 | `std::unordered_map` | 哈希表 |
| `#define MAX(a,b) ((a)>(b)?(a):(b))` | `template<T> T max(T a, T b)` | 泛型 |
| `pthread_create` | `std::thread` | 多线程 |
| `pthread_mutex_lock` | `std::lock_guard<std::mutex>` | 锁 |
| Makefile | CMakeLists.txt | 构建 |
