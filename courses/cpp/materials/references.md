## 学习顺序建议

1. 先通读 runoob.com 的 C++ 教程（第一层到第三层）
2. 遇到概念不清时查 cppreference.com
3. 学完第三层后开始做 CMake + 多文件项目
4. 学完第四层后开始接触嵌入式 Linux 和 AI 推理框架
