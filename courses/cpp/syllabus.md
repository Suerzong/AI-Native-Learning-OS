# C++ 编程与嵌入式 Linux 工程准备 课程大纲

本课程共 42 节，面向嵌入式 Linux、机器人和 AI 推理部署。从 C 过渡到 C++，按四层递进结构组织。

---

# 课程架构总览

```
第一层：C++ 基础篇（1-10节）       → 从 C 过渡到 C++，能用 C++ 写出 C 能写的程序
第二层：面向对象编程篇（11-22节）  → 能用类和继承组织代码
第三层：泛型编程与 STL 篇（23-32节） → 能用模板和 STL 提高代码复用
第四层：工程实战与嵌入式准备篇（33-42节） → 能组织工程、交叉编译、为 AI 部署做准备
```

---

# 第一层：C++ 基础篇（第 1-10 节）

目标：完成从 C 到 C++ 的过渡。解决"怎么搭建 C++ 开发环境、C++ 的基本语法和 C 有什么不同、C++ 新增了哪些好用的特性"。

| 节号 | 内容 | 核心能力 | 前置依赖 |
|------|------|---------|---------|
| 01 | 课程介绍与 C/C++ 关系 | 了解 C++ 的定位、与 C 的关系、学习路径 | C 语言基础 |
| 02 | 开发环境与 Hello World | g++ 编译、VS Code 配置、第一个 C++ 程序 | 01 |
| 03 | 数据类型新特性 | bool、auto、decltype、constexpr、nullptr | 02 |
| 04 | const 与类型系统 | const 在 C++ 中的增强语义、类型别名 using | 03 |
| 05 | 运算符与 new/delete | 作用域运算符、new/delete vs malloc/free | 03 |
| 06 | 条件语句新特性 | if constexpr、if 初始化语句 | 04 |
| 07 | 循环与 range-based for | 基于范围的 for 循环、auto 在循环中 | 06 |
| 08 | 函数新特性（上） | 默认参数、引用传参 | 05 |
| 09 | 函数新特性（下） | 函数重载、inline 函数 | 08 |
| 10 | 数组与 string 类 | std::string、std::array、C 字符串 vs C++ 字符串 | 08 |

## 阶段一掌握标准

能用 C++ 写出一个命令行程序，使用 `string`、`cin`/`cout`、函数重载、引用传参、range-based for 循环。能逐行解释每个 C++ 新特性的作用和等价的 C 写法。

# 故障排查专题

以下内容不按编号线型学习，而是在遇到问题时回看：

- **编译错误排查**：常见编译错误的诊断思路（Lesson 2、23-24）
- **链接错误排查**：undefined reference、multiple definition（Lesson 39-40）
- **内存问题排查**：内存泄漏、野指针、double free（Lesson 5、31-32）
- **模板编译错误**：SFINAE、模板实例化失败（Lesson 23-26）
- **交叉编译问题**：库路径、工具链配置（Lesson 41）
