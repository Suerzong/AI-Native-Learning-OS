### 练习 001: 变量与类型转换

- **目标技能**: basics — 变量与数据类型
- **难度**: Level 1
- **类型**: 编写
- **题目**:
  编写一个程序，完成以下任务：
  1. 从用户输入获取一个数字字符串（如 "3.14"）
  2. 将其转换为浮点数
  3. 计算其平方并保留 2 位小数
  4. 输出结果，格式为："3.14 的平方是 9.86"

- **成功标准**:
  - 使用 `input()` 获取输入
  - 使用 `float()` 进行类型转换
  - 使用 `**` 运算符计算平方
  - 使用 f-string 格式化输出
  - 结果保留 2 位小数

- **常见错误**:
  - 忘记将 input() 的结果转换为数字类型
  - 混淆 `=`（赋值）和 `==`（比较）
  - 使用 `round()` 但不理解其行为（银行家舍入）

### 练习 002: 字符串切片与方法

- **目标技能**: basics — 字符串操作
- **难度**: Level 1
- **类型**: 编写
- **题目**:
  给定一个邮箱地址字符串 `email = "student@bupt.edu.cn"`，编写代码：
  1. 提取用户名部分（@ 之前）
  2. 提取域名部分（@ 之后）
  3. 将用户名转换为大写
  4. 输出格式："用户名: STUDENT, 域名: bupt.edu.cn"

- **成功标准**:
  - 使用 `split("@")` 分割字符串
  - 使用 `.upper()` 转换大小写
  - 使用 f-string 格式化输出

- **常见错误**:
  - 使用索引硬编码位置（如 email[0:7]），不够通用
  - 忘记 split() 返回的是列表

### 练习 003: 条件语句 — 成绩等级

- **目标技能**: control-flow — 条件语句
- **难度**: Level 1
- **类型**: 编写
- **题目**:
  编写一个成绩等级判断程序：
  - 90-100: A
  - 80-89: B
  - 70-79: C
  - 60-69: D
  - 0-59: F
  - 其他: 输入无效

  要求使用 if/elif/else 实现，并处理边界情况。

- **成功标准**:
  - 使用 `elif` 而非嵌套 `if`
  - 处理 < 0 和 > 100 的输入
  - 使用链式比较 `90 <= score <= 100`
  - 输出包含分数和等级

- **常见错误**:
  - 使用嵌套 if 导致逻辑混乱
  - 边界值处理不当（如 score = 90 时可能落入 B 级）
  - 忘记处理非数字输入

### 练习 004: 循环 — 九九乘法表

- **目标技能**: control-flow — 循环
- **难度**: Level 2
- **类型**: 编写
- **题目**:
  使用嵌套 for 循环打印九九乘法表，格式如下：
  ```
  1×1=1
  1×2=2  2×2=4
  1×3=3  2×3=6  3×3=9
  ...
  ```
  要求使用 `range()` 和 `print()` 的 `end` 参数控制格式。

- **成功标准**:
  - 使用两层嵌套 for 循环
  - 外层循环控制行（1-9）
  - 内层循环控制列（1 到当前行数）
  - 使用 `end="\t"` 或格式化控制对齐
  - 每行结束时换行

- **常见错误**:
  - range() 的起止值搞错（range 默认从 0 开始）
  - print() 默认换行导致格式混乱
  - 内层循环范围错误

### 练习 005: 列表推导式与数据处理

- **目标技能**: data-structures — 推导式
- **难度**: Level 2
- **类型**: 编写
- **题目**:
  给定一个包含学生成绩的列表：
  ```python
  scores = [85, 92, 78, 95, 60, 88, 73, 91, 67, 84]
  ```
  使用列表推导式完成以下任务（不使用普通 for 循环）：
  1. 筛选出所有 >= 80 的成绩
  2. 将每个成绩转换为等级（>=90 为 "A"，>=80 为 "B"，>=70 为 "C"，其他为 "D"）
  3. 计算及格成绩（>=60）的平均分
  4. 找出最高分和最低分的索引位置

- **成功标准**:
  - 使用列表推导式（不是普通 for 循环）
  - 使用条件表达式（三元运算符）
  - 使用 `sum()` / `len()` 计算平均值
  - 使用 `max()` / `min()` 和 `.index()` 找位置

- **常见错误**:
  - 推导式中条件位置错误（`[x for x in lst if cond]` 而非 `[x if cond for x in lst]`）
  - 三元表达式和推导式中的条件过滤混淆
  - 列表为空时计算平均值会报错

### 练习 006: 字典操作 — 词频统计

- **目标技能**: data-structures — 字典
- **难度**: Level 2
- **类型**: 编写
- **题目**:
  给定一段英文文本：
  ```python
  text = "the quick brown fox jumps over the lazy dog the fox"
  ```
  编写程序统计每个单词出现的次数，并按频率从高到低输出。

- **成功标准**:
  - 使用 `split()` 将文本拆分为单词列表
  - 使用字典统计词频（手动实现，不直接用 Counter）
  - 使用 `sorted()` 配合 `key` 参数排序
  - 输出格式：`"the: 3"` 等

- **常见错误**:
  - 忘记处理大小写（本题均为小写，但应该思考这个问题）
  - 排序时不知道如何按键或值排序
  - 使用不存在的键直接赋值会报错（应用 setdefault 或 get）

### 练习 007: 元组解包与函数返回

- **目标技能**: data-structures — 元组
- **难度**: Level 1
- **类型**: 编写
- **题目**:
  编写一个函数 `min_max_avg(lst)`，接收一个数字列表，返回一个元组 `(最小值, 最大值, 平均值)`。
  调用该函数并使用元组解包获取结果。

- **成功标准**:
  - 函数返回元组
  - 调用时使用元组解包：`mn, mx, avg = min_max_avg(scores)`
  - 处理空列表的情况（返回 None 或抛出异常）
  - 平均值保留 2 位小数

- **常见错误**:
  - 返回值数量和解包变量数量不一致
  - 空列表时没有处理导致除零错误
  - 混淆 return 多个值（实际是返回元组）
