# concepts.md — Python 核心概念索引

## Layer 1: Python 基础概念

### 环境与基础
- **Python 解释器**：将 Python 代码转换为机器码并执行的程序。CPython 是最常用的实现。
- **REPL**：Read-Eval-Print Loop，交互式编程环境。在命令行输入 `python` 进入。
- **脚本文件**：以 `.py` 为扩展名的文本文件，由解释器逐行执行。
- **缩进**：Python 使用 4 个空格缩进来定义代码块，替代其他语言的大括号。

### 变量与数据类型
- **动态类型**：变量不需要声明类型，类型由赋值决定，可以在运行时改变。
- **不可变类型**：int, float, str, tuple —— 创建后不能修改。
- **可变类型**：list, dict, set —— 创建后可以修改。
- **None**：Python 的空值类型，表示"无"或"空"。用 `is None` 判断。
- **类型标注**：Python 3.5+ 支持类型提示（Type Hints），如 `x: int = 10`，但不强制执行。

### 运算符
- **算术运算符**：`+`, `-`, `*`, `/`, `//`（整除）, `%`（取模）, `**`（幂运算）
- **比较运算符**：`==`, `!=`, `>`, `<`, `>=`, `<=`
- **逻辑运算符**：`and`, `or`, `not` —— 支持短路求值
- **身份运算符**：`is`, `is not` —— 比较对象身份（内存地址）
- **成员运算符**：`in`, `not in` —— 检查元素是否在序列中

### 字符串
- **不可变性**：字符串一旦创建就不能修改，任何"修改"操作都返回新字符串。
- **索引与切片**：`s[0]`, `s[1:3]`, `s[::-1]`（反转）
- **常用方法**：split(), join(), strip(), replace(), find(), startswith(), endswith(), upper(), lower()
- **f-string**：Python 3.6+ 的格式化字符串，如 `f"Hello, {name}!"`

### 条件语句
- **Truthy/Falsy**：在布尔上下文中，以下值为 False：0, 0.0, '', [], {}, set(), None, False
- **三元表达式**：`x if condition else y`
- **链式比较**：`1 < x < 10` 等价于 `1 < x and x < 10`

### 循环
- **for 循环**：遍历可迭代对象。`for item in iterable:`
- **while 循环**：条件为 True 时重复执行。
- **range()**：生成整数序列。`range(start, stop, step)`
- **enumerate()**：同时获取索引和值。`for i, v in enumerate(lst):`
- **break/continue**：break 跳出循环，continue 跳到下一次迭代。
- **else 子句**：循环正常结束（非 break）时执行。

### 数据结构
- **列表（list）**：有序、可变、可重复。用 `[]` 创建。
- **元组（tuple）**：有序、不可变、可重复。用 `()` 创建。
- **字典（dict）**：无序（Python 3.7+ 保持插入顺序）、可变、键不可重复。用 `{}` 创建。
- **集合（set）**：无序、可变、元素不可重复。用 `set()` 或 `{}` 创建。
- **推导式**：用一行代码生成列表/字典/集合的语法糖。

## Layer 2: 函数与面向对象概念

### 函数
- **函数是对象**：Python 中函数是一等公民，可以赋值给变量、作为参数传递、作为返回值。
- **参数传递**：Python 使用"对象引用传递"（pass by object reference）。不可变类型看起来像值传递，可变类型看起来像引用传递。
- ***args 和 **kwargs**：`*args` 收集位置参数为元组，`**kwargs` 收集关键字参数为字典。
- **文档字符串（docstring）**：函数定义后的第一个字符串，用于文档说明。

### 作用域
- **LEGB 规则**：Local（局部） → Enclosing（外层函数） → Global（全局） → Built-in（内置）
- **global 关键字**：在函数内修改全局变量。
- **nonlocal 关键字**：在嵌套函数中修改外层函数的变量。

### 闭包与 Lambda
- **闭包**：一个函数捕获了外层函数的变量（自由变量），即使外层函数已返回。
- **lambda**：匿名函数，语法：`lambda 参数: 表达式`。适用于简单的回调函数。

### 装饰器
- **装饰器本质**：接受函数作为参数并返回新函数的高阶函数。
- **@语法糖**：`@decorator` 等价于 `func = decorator(func)`。
- **functools.wraps**：保留被装饰函数的元信息（名称、文档字符串等）。

### 类与对象
- **类**：对象的蓝图/模板，定义属性和方法。
- **实例**：根据类创建的具体对象。
- **self**：实例方法的第一个参数，指向调用该方法的实例。
- **类属性 vs 实例属性**：类属性属于类本身，实例属性属于各个实例。
- **继承**：子类继承父类的属性和方法。使用 `super()` 调用父类方法。
- **MRO**：Method Resolution Order，多继承时方法查找顺序。C3 线性化算法。
- **多态**：不同类的对象对同一方法有不同的实现。

### 魔法方法
- **__init__**：构造方法，初始化实例属性。
- **__str__**：面向用户的字符串表示，`print(obj)` 时调用。
- **__repr__**：面向开发者的字符串表示，`repr(obj)` 或交互式环境中显示。
- **__len__**：`len(obj)` 时调用。
- **__eq__**：`==` 比较时调用。
- **__lt__/__gt__**：`<` / `>` 比较时调用。
- **__getitem__**：`obj[key]` 索引时调用。
- **__enter__/__exit__**：with 语句的上下文管理协议。

## Layer 3: 实用技能概念

### 模块与包
- **模块**：一个 `.py` 文件就是一个模块。
- **包**：包含 `__init__.py` 的目录。
- **import 搜索路径**：当前目录 → PYTHONPATH → 标准库 → site-packages。
- **__name__=="__main__"**：当模块被直接运行时为 True，被 import 时为模块名。

### 文件 I/O
- **with 语句**：自动管理资源，确保文件正确关闭。
- **文件模式**：r（读）、w（写，覆盖）、a（追加）、b（二进制）、+（读写）。

### 异常处理
- **异常层次结构**：BaseException → Exception → 具体异常（ValueError, TypeError 等）
- **try/except/else/finally**：else 在无异常时执行，finally 始终执行。
- **自定义异常**：继承 Exception 类。

### 正则表达式
- **re.match()**：从字符串开头匹配。
- **re.search()**：在字符串中搜索第一个匹配。
- **re.findall()**：返回所有匹配的列表。
- **re.sub()**：替换匹配的内容。
- **分组**：用 `()` 捕获匹配的子串。

## Layer 4: 进阶概念

### 生成器
- **惰性求值**：按需生成值，不一次性占用内存。
- **yield**：暂停函数执行并返回值，下次调用时从暂停处继续。
- **生成器表达式**：`(x for x in range(10))`，类似列表推导式但返回生成器。

### 迭代器
- **迭代器协议**：实现 `__iter__()` 和 `__next__()` 方法。
- **StopIteration**：迭代器耗尽时抛出的异常。

### 上下文管理器
- **协议**：实现 `__enter__()` 和 `__exit__()` 方法。
- **contextlib.contextmanager**：用装饰器简化上下文管理器的编写。

### 并发
- **GIL**：Global Interpreter Lock，CPython 的全局锁，限制同一时刻只有一个线程执行 Python 字节码。
- **threading**：适用于 I/O 密集型任务。
- **multiprocessing**：适用于 CPU 密集型任务，每个进程有独立的 GIL。

### NumPy
- **ndarray**：N 维数组对象，支持向量化操作。
- **广播（Broadcasting）**：不同形状的数组进行运算时的自动扩展规则。
- **向量化**：用数组操作替代循环，大幅提升性能。

### Pandas
- **Series**：一维带标签数组。
- **DataFrame**：二维带标签数据结构，类似电子表格。
- **索引与筛选**：loc（标签）、iloc（位置）、布尔索引。
