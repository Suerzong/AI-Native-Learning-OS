#### 题目 1: 概念题
- **题目**: Python 中 `is` 和 `==` 有什么区别？分别在什么场景下使用？
- **预期答案要点**:
  - `==` 比较值是否相等（调用 `__eq__` 方法）
  - `is` 比较两个变量是否指向同一个对象（内存地址相同）
  - 使用场景：`is` 用于判断 None、True、False；`==` 用于判断值相等
  - `a is b` 意味着 `a == b`，但 `a == b` 不意味着 `a is b`
- **得分**: [0-25]

#### 题目 2: 代码定位题
- **题目**: 以下代码有什么问题？会导致什么结果？
```python
data = [1, 2, 3, 4, 5]
for i in range(len(data)):
    if data[i] > 3:
        del data[i]
print(data)
```
- **预期答案**:
  - 在循环中删除列表元素导致索引偏移
  - 当 i=3 时删除 data[3]=4，列表变为 [1,2,3,5]
  - 此时 i=4，但 data[4] 不存在（越界）或指向错误的元素
  - 正确做法：使用列表推导式 `data = [x for x in data if x <= 3]`
- **得分**: [0-25]

#### 题目 3: 修改题
- **题目**: 以下代码将所有单词首字母大写。请修改为只大写偶数索引位置的单词（索引从 0 开始）。
```python
words = ["hello", "world", "python", "is", "great"]
result = [w.capitalize() for w in words]
print(result)
```
- **预期修改**:
```python
words = ["hello", "world", "python", "is", "great"]
result = [w.upper() if i % 2 == 0 else w for i, w in enumerate(words)]
print(result)
```
- **得分**: [0-25]

#### 题目 4: 调试题
- **题目**: 以下函数旨在返回列表中所有正数的平均值，但有 bug。找出并修复。
```python
def avg_positive(numbers):
    pos = [n for n in numbers if n > 0]
    return sum(pos) / len(pos)

print(avg_positive([]))  # 报错！
print(avg_positive([-1, -2, -3]))  # 也报错！
```
- **预期修复**:
```python
def avg_positive(numbers):
    pos = [n for n in numbers if n > 0]
    if not pos:
        return 0  # 或返回 None，或抛出异常
    return sum(pos) / len(pos)
```
- **得分**: [0-25]
