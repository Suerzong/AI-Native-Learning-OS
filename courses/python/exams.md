#### 题目列表

**一、概念题（每题 5 分，共 25 分）**

1. 解释 Python 中"动态类型"的含义，并举例说明变量类型可以在运行时改变。

2. 列表（list）、元组（tuple）、集合（set）三者有什么区别？各适用于什么场景？

3. 什么是"短路求值"？`and` 和 `or` 的短路规则分别是什么？

4. 解释列表推导式 `[x**2 for x in range(10) if x % 2 == 0]` 每一部分的含义。

5. Python 中 `is` 和 `==` 的区别是什么？什么情况下两者结果可能不同？

**二、代码阅读题（每题 5 分，共 15 分）**

6. 以下代码的输出是什么？解释原因。
```python
a = [1, 2, 3]
b = a
b.append(4)
print(a)
```

7. 以下代码的输出是什么？
```python
d = {"a": 1, "b": 2, "c": 3}
print(d.get("d", 0))
print(d["d"])
```

8. 以下代码的输出是什么？
```python
x = 10
result = "even" if x % 2 == 0 else "odd"
print(result)
```

**三、代码编写题（每题 10 分，共 30 分）**

9. 编写一个程序，接收用户输入的字符串，统计其中字母、数字、空格和其他字符的个数。

10. 使用字典推导式，创建一个字典，键为 1-10 的数字，值为该数字的平方。

11. 给定一个嵌套列表 `data = [[1, 2], [3, 4], [5, 6]]`，使用列表推导式将其展平为 `[1, 2, 3, 4, 5, 6]`。

**四、调试题（每题 10 分，共 30 分）**

12. 以下代码有什么问题？修复它。
```python
def swap(a, b):
    a, b = b, a

x = 1
y = 2
swap(x, y)
print(x, y)  # 预期输出 2 1，实际输出 1 2
```

13. 以下代码在处理空列表时会崩溃，修复它。
```python
numbers = []
average = sum(numbers) / len(numbers)
print(average)
```

14. 以下代码的输出不符合预期，找出问题。
```python
text = "hello world"
words = text.split()
result = ""
for word in words:
    result += word.capitalize() + " "
print(result)  # 预期 "Hello World"，但末尾多了一个空格
```

- 代码编写题：功能正确得 7 分，代码质量好再得 3 分
- 调试题：找到问题得 5 分，修复正确再得 5 分
