# 一、数据结构基本概念与算法复杂度

## (一) 判断题

**1.** 对应同一个数据结构，可以有不同的实现方法。
**答案：T**

**2.** 抽象数据类型中，描述数据类型的方法与实现操作的算法和编程语言有关。
**答案：F**

**3.** 解决问题的效率，跟数据的组织方式无关。
**答案：F**

**4.** 数据的逻辑结构是指数据的各数据项之间的逻辑关系。
**答案：F**（逻辑结构是指数据元素之间的关系，而非数据项）

**5.** 数据结构包括数据对象集以及它们的逻辑结构和物理结构，还包括与数据对象相关联的操作集，以及实现这些操作的高效的算法。

**8.** 算法的时间复杂度指的是算法所需时间随问题规模增长的增长率。
**答案：T**

**9.** 度量算法的时间复杂度可以通过分析算法的基本操作执行次数来实现。
**答案：T**

**10.** 多项式时间算法指的是算法的时间复杂度是多项式级别的，可以表示为一个多项式函数，即 O(n^k)，k≥0。

## (三) 编程题

**1. 用指针交换两个主函数变量数值**

题目：调用子函数swap，完成主函数两个整型变量a和b的交换。

答案：
```c
void swap(int *data1, int *data2) {
    int temp = *data1;
    *data1 = *data2;
    *data2 = temp;
}
```

**2. 按等级统计学生成绩**

题目：实现set_grade函数，根据成绩设置等级（85-100为A，70-84为B，60-69为C，0-59为D），返回不及格人数。

答案：
```c
int set_grade(struct student *p, int n) {
    int i, count = 0;
    for (i = 0; i < n; i++) {
        if (p[i].score >= 85) p[i].grade = 'A';
        else if (p[i].score >= 70) p[i].grade = 'B';
        else if (p[i].score >= 60) p[i].grade = 'C';
        else { p[i].grade = 'D'; count++; }
    }
    return count;
}
```

## (三) 编程题

**1. 求链表的倒数第m个元素**

题目：在不改变链表的前提下，求链式存储的线性表的倒数第m（>0）个元素。

答案（双指针法）：
```c
ElementType Find(List L, int m) {
    PtrToNode fast = L->Next, slow = L->Next;
    int i;
    for (i = 0; i < m; i++) {
        if (fast == NULL) return ERROR;
        fast = fast->Next;
    }
    while (fast != NULL) {
        fast = fast->Next;
        slow = slow->Next;
    }
    return slow->Data;
}
```

**2. 线性表元素的区间删除**

题目：删除顺序存储线性表中所有值大于min且小于max的元素，剩余元素保持相对位置不变。

答案：
```c
List Delete(List L, ElementType minD, ElementType maxD) {
    int i, j = 0;
    for (i = 0; i <= L->Last; i++) {
        if (L->Data[i] <= minD || L->Data[i] >= maxD)
            L->Data[j++] = L->Data[i];
    }
    L->Last = j - 1;
    return L;
}
```

**3. 单链表逆转**

题目：实现一个函数，将给定的单链表逆转。

答案：
```c
List Reverse(List L) {
    PtrToNode old_head = L;
    PtrToNode new_head = NULL;
    PtrToNode temp;
    while (old_head) {
        temp = old_head->Next;
        old_head->Next = new_head;
        new_head = old_head;
        old_head = temp;
    }
    return new_head;
}
```

**4. 数组循环左移**

题目：将数组a中n个整数循环向左移m个位置，要求移动次数尽量少。

答案（三次翻转法）：
```c
void Reverse(int a[], int left, int right) {
    int temp;
    for (; left < right; left++, right--) {
        temp = a[left];
        a[left] = a[right];
        a[right] = temp;
    }
}
// 在main中：m = m % n; Reverse(a, 0, m-1); Reverse(a, m, n-1); Reverse(a, 0, n-1);
```

## (三) 编程题

**1. 递归计算Ackermann函数**

题目：实现Ackermann函数的计算。

答案：
```c
int Ack(int m, int n) {
    if (m == 0) return n + 1;
    if (n == 0) return Ack(m - 1, 1);
    return Ack(m - 1, Ack(m, n - 1));
}
```

**2. 仅有头指针的队列**

题目：用循环数组实现队列，只设front和count，实现判空、入队、出队。

答案：
```c
bool IsEmpty(Queue queue) {
    return queue->count == 0;
}
void EnQueue(Queue queue, QElemSet x) {
    if (queue->count == queue->capacity) {
        printf("ERROR\n");
        return;
    }
    Position rear = (queue->front + queue->count) % queue->capacity;
    queue->data[rear] = x;
    queue->count++;
}
void DeQueue(Queue queue) {
    if (IsEmpty(queue)) {
        printf("ERROR\n");
        return;
    }
    queue->front = (queue->front + 1) % queue->capacity;
    queue->count--;
}
```

**3. 在一个数组中实现两个栈**

题目：在长度为n的数组中实现两个栈，使得二者在元素总数为n之前都不溢出。

答案：
```c
void InitStack(Stack stack, int n) {
    stack->capacity = n;
    stack->data = (SElemSet *)malloc(n * sizeof(SElemSet));
    stack->top1 = -1;
    stack->top2 = n;
}
bool Push(Stack stack, int tag, SElemSet x) {
    if (stack->top1 + 1 == stack->top2) {
        printf("Stack Full\n");
        return false;
    }
    if (tag == 1) stack->data[++(stack->top1)] = x;
    else stack->data[--(stack->top2)] = x;
    return true;
}
bool IsEmpty(Stack stack, int tag) {
    if (tag == 1) return stack->top1 == -1;
    else return stack->top2 == stack->capacity;
}
SElemSet TopPop(Stack stack, int tag) {
    if (tag == 1) {
        if (stack->top1 == -1) return ERROR;
        return stack->data[(stack->top1)--];
    } else {
        if (stack->top2 == stack->capacity) return ERROR;
        return stack->data[(stack->top2)++];
    }
}
```
