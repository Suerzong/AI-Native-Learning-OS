## 第二章：线性表

### 判断题（错 2/9）

- [ ] **5.** 带头结点单循环链表，任一结点后继指针域均不空。你答 F → **T**（尾结点 next 指向头结点，始终不空）

### 选择题（错 4/18）

- [ ] **6.** 两表头尾连接 O(1)。你答 D(带头双循环) → **C(带尾指针的单循环)**。带尾指针才能 O(1) 找到尾结点
- [ ] **13.** 双向链表 p 后插 s，已执行 s->next=p->next; p->next=s。你答 B → **C**。p->next 已变，必须先保存原后继
- [ ] **14.** 删除递增有序表中值在(min,max)区间的元素 → **D**
- [ ] **18.** q 被摘下后插到头结点后 → **D**（q 移到头结点后成为首元素）

### 编程题薄弱点

- [ ] **区间删除** — 读写双指针，不是找 minD/maxD 位置
- [ ] **单链表逆转** — 三指针 prev/curr/next，不是用栈
- [ ] **数组循环左移** — 三次翻转法，不是逐个移动
