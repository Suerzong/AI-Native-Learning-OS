typedef struct PolyNode *Polynomial; //表示多项式的链表结构
struct PolyNode {
int coef;
int expon;
Polynomial link;
}
这段代码定义了一个用于表示多项式的链表结构。让我们逐步解析这段代码的意义：
1. typedef struct PolyNode *Polynomial;
typedef 是 C 语言中的一个关键字，用于为现有的数据类型定义一个新的名字。
struct PolyNode * 表示一个指向 struct PolyNode 结构体的指针。
Polynomial 是这个指针类型的新名字。通过这个定义，Polynomial 现在可以用来声明指向 struct PolyNode 结构体的指针变量。

2. struct PolyNode { ... }
struct 是 C 语言中的关键字，用于定义一个结构体类型。
PolyNode 是这个结构体的名字。

3. 结构体成员
int coef;
coef 是结构体的一个成员，类型为 int。它用于存储多项式项的系数。
int expon;
expon 是结构体的另一个成员，类型为 int。它用于存储多项式项的指数。
Polynomial link;
link 是结构体的第三个成员，类型为 Polynomial，即指向 struct PolyNode 的指针。它用于指向下一个多项式项，从而形成一个链表。

总结
这段代码定义了一个链表结构，用于表示多项式。每个节点（PolyNode）包含两个整数成员 coef 和 expon，分别表示多项式项的系数和指数，以及一个指向下一个节点的指针 link。通过这种方式，可以方便地表示和操作多项式，例如多项式的加法、乘法等操作。
