# 核心概念索引

本文件记录学习者需要掌握的关键概念。每个概念都应能回答三个问题：它是什么、为什么需要它、在代码里怎么实现。

## 基础概念

| 概念 | 简要说明 | 对应节号 |
|------|---------|---------|
| 时间复杂度 | 算法执行时间随输入规模增长的变化趋势，用大 O 表示 | 02 |
| 空间复杂度 | 算法占用内存随输入规模增长的变化趋势 | 02 |
| 大 O 表示法 | 描述算法复杂度上界的数学符号 | 02 |
| 最好/最坏/平均情况 | 算法在不同输入下的性能表现 | 02 |
| 渐进分析 | 忽略常数因子和低阶项，关注增长趋势 | 02 |

## 线性结构

| 概念 | 简要说明 | 对应节号 |
|------|---------|---------|

| 双向链表 | 节点有 prev 和 next 两个指针 | 06 |
| 循环链表 | 尾节点的 next 指向头节点 | 06 |
| 哨兵节点（头节点） | 简化边界处理的虚拟头节点 | 04 |

| 队列 | FIFO（先进先出）结构 | 09-10 |
| 循环队列 | 用数组+取模实现的固定大小队列 | 09 |
| 双端队列 | 两端都可插入和删除 | 10 |

## 算法思维

| 概念 | 简要说明 | 对应节号 |
|------|---------|---------|
| 递归 | 函数调用自身，将问题分解为更小的子问题 | 35 |
| 分治 | 将问题分成多个子问题，分别求解再合并 | 35 |
| 贪心 | 每步选择局部最优，期望达到全局最优 | 36 |
| 动态规划 DP | 通过子问题的最优解构建原问题的最优解 | 37-38 |
| 重叠子问题 | 递归中反复计算同一子问题 | 37 |
| 最优子结构 | 问题的最优解包含子问题的最优解 | 36-37 |
| 状态转移方程 | 描述 DP 中状态之间关系的数学表达式 | 37-38 |
| 环形缓冲区 | 嵌入式常用的固定大小 FIFO 缓冲区 | 39 |
| 位图 Bitmap | 用 bit 位表示状态，空间效率极高 | 39 |
