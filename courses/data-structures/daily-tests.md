## 算法分析样例

1. 以下代码的时间复杂度是什么？`for (int i = 1; i < n; i *= 2) sum++;`
2. 以下递归函数的时间复杂度是什么？
   ```c
   int f(int n) {
       if (n <= 0) return 0;
       return f(n-1) + f(n-1);
   }
   ```
3. 空间复杂度为 O(1) 是什么意思？请举一个例子。
4. 以下代码中，`arr` 有 n 个元素，`find` 函数的最好和最坏时间复杂度分别是多少？
   ```c
   int find(int arr[], int n, int target) {
       for (int i = 0; i < n; i++)
           if (arr[i] == target) return i;
       return -1;
   }
   ```

## 链表样例

1. 单链表和数组相比，随机访问的时间复杂度分别是多少？
2. 写出在单链表头部插入新节点的代码片段（假设已有 head 指针）。
3. 以下代码试图删除链表中值为 val 的第一个节点，找出 bug：
   ```c
   void deleteNode(ListNode* head, int val) {
       ListNode* curr = head;
       while (curr != NULL) {
           if (curr->val == val) {
               curr = curr->next;
               free(curr);
               return;
           }
           curr = curr->next;
       }
   }
   ```
4. 反转链表时，为什么需要三个指针（prev, curr, next）而不是两个？
