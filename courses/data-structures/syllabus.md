# 数据结构与算法 课程大纲

本课程共 42 节，按四层递进结构组织。每节围绕一个核心知识点，包含概念讲解、C/C++ 实现、复杂度分析和练习。

---

# 课程架构总览

```
第一层：基础概念与线性结构（1-12节）
  复杂度分析 → 数组 → 链表 → 栈 → 队列
  目标：掌握基本数据结构，能用 C/C++ 实现并分析复杂度

第二层：树与哈希（13-22节）
  哈希表 → 二叉树 → BST → 堆 → 优先队列
  目标：掌握非线性数据结构，理解树的递归本质

第三层：图与排序算法（23-34节）
  图 → 基础排序 → 高级排序 → 查找算法
  目标：掌握常用排序和图的基本算法

第四层：综合应用与算法思维（35-42节）
  递归/分治 → 贪心 → 动态规划 → 嵌入式场景选型 → 刷题策略
  目标：形成算法思维，能针对嵌入式场景选择合适的数据结构
```
