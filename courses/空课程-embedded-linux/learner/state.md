## 高发弱点预判

### 权限模型

- 症状：遇到 "Permission denied" 就 sudo，不理解为什么被拒绝
- 根因：不理解 rwx + owner/group/others 三层模型
- 纠正方法：每次 Permission denied 先 ls -l 看权限

### 路径思维

- 症状：脚本中相对路径在手动执行时正常，但 cron 中失败
- 根因：不理解"当前工作目录"依赖于执行上下文
- 纠正方法：脚本中所有路径用绝对路径，或显式 cd 到目标目录

### 发行版差异

- 症状：在 CentOS 教程中看到 yum 就在 Ubuntu 上敲 yum
- 根因：不了解不同发行版的包管理差异
- 纠正方法：建立概念映射表（yum→apt, rpm→dpkg）

### vim 模式

- 症状：进入 vim 后不知道自己在哪个模式
- 根因：vim 的模态编辑和所有现代编辑器不同
- 纠正方法：强制先按 Esc 回到命令模式，再想下一步操作
