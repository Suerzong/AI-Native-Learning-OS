# 练习库

所有练习按 鸟哥 章节编号。格式：`Ch{章}-{类型}-{序号}`

类型：CONCEPT（概念题）、COMMAND（命令操作题）、SCRIPT（脚本编程题）、DEBUG（故障排查题）、SCENARIO（场景选择题）

出题前必须先判断任务是否属于低价值题。只考死记命令选项的题不应进入本任务库。涉及命令时，除非选参数本身能帮助理解设计逻辑，否则直接给出提示，把练习重点放在命令选择、场景判断、错误排查和相近命令区别上。

## 任务模板

```md
## 任务名称

- 价值判断：这道题为什么不是低价值机械题？
- 目标技能：
- 起始材料：
- 任务要求：
- 限制条件：
- 成功标准：
- 常见错误：
- 反馈重点：
```
