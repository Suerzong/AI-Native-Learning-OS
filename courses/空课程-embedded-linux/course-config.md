# 课程身份

- 课程名：嵌入式 Linux 系统
- 当前状态：空课程
- 目标：掌握 Linux 基本操作、系统编程、交叉编译、设备文件访问和板端调试，为 Edge AI 部署打基础
- 前置知识：C/C++ 基础、计算机系统基础、STM32/MCU 经验
- 对应能力模块：模块 8（嵌入式 Linux 系统）、模块 2（编程语言与软件基础）、模块 3（计算机系统基础）、模块 11（模型部署与端侧优化）、模块 12（工程项目与科研能力）
- 课程规模：待定，建议 42 节
- 课程定位：从 PC 端编程过渡到板端 Linux 工程，为 Jetson/RK3588/树莓派等平台上的 AI 推理应用做准备

# 课程架构模板

```
第一层：Linux 基础
  文件系统 -> 权限 -> 进程 -> 包管理 -> Shell

第二层：构建与部署
  gcc/g++ -> Makefile -> CMake -> 交叉编译 -> SSH/SCP

第三层：系统编程与外设访问
  文件 I/O -> 线程/进程 -> Socket -> 串口/I2C/SPI -> 日志

第四层：边缘 AI 工程衔接
  摄像头采集 -> OpenCV -> 推理运行时 -> 性能分析 -> 项目部署
```
