# NNDL 每日诊断短测

每日短测只用于判断今天该补哪一个 NNDL 小节，不用于制造额外内容。

## 出题边界

- 只考已讲 NNDL 小节和已解锁前置。
- 不考外部资料、代码实现、框架用法、工程部署。
- 每题必须标注来源和未学内容检查。
- “了解”层内容不进入正式短测。

## 模板

```md
## YYYY-MM-DD

### 诊断题

1. 题目：
   - 考察来源：
   - 未学内容检查：
2. 题目：
   - 考察来源：
   - 未学内容检查：
3. 题目：
   - 考察来源：
   - 未学内容检查：

### 结果

- 正确率：
- 主要错误：
- 今日训练目标：
- 是否允许推进：
```
