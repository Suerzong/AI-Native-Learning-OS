每日短测用于判断今天应该训练什么，而不是为了打分。

## 感知机样例

1. 感知机的输出公式是什么？写出 y = f(Σ w_i * x_i + b) 中 f 是什么函数。
2. 给定 w=[0.5, 0.5], b=-0.7，输入 x=[1, 1]，用阶跃函数激活，输出是什么？
3. 用 NumPy 写出感知机前向传播的代码（3 行以内）。
4. 为什么 XOR 不能用单层感知机解决？画图说明。

---

## 激活函数样例

1. Sigmoid 的输出范围是什么？它的导数在 z=0 时等于多少？
2. 写出 ReLU 的数学表达式。当 z=-3 时，ReLU 及其导数分别是什么？
3. 用 NumPy 实现 Softmax，输入 [2.0, 1.0, 0.1]，输出概率。
4. 如果一个 10 层网络全用 Sigmoid，梯度反向传播到第一层会怎样？为什么？

---

## 前向传播样例

1. 一个全连接层的前向传播公式是什么？写出矩阵形式。
2. 给定输入 x shape=(1, 3)，权重 W shape=(3, 2)，偏置 b shape=(1, 2)，输出的 shape 是什么？
3. 用 NumPy 写出一行代码实现 `z = W @ x + b` 的前向传播。
4. 如果 W 的 shape 是 (2, 3) 但 x 的 shape 是 (1, 2)，会报什么错？为什么？

---

## 反向传播样例

1. 链式法则的公式是什么？如果 L = f(g(x))，dL/dx = ?
2. 对于 z = Wx + b，∂L/∂W 的维度和 W 的维度是什么关系？
3. 用 NumPy 写出计算 dW = dz @ x.T 的代码。
4. 如果某一层的梯度是 NaN，可能的原因有哪些？列出至少 2 个。

---

## 梯度下降样例

1. BGD、SGD、Mini-batch GD 各自每次用多少数据计算梯度？
2. SGD 相比 BGD 的优缺点各是什么？
3. 如果学习率设为 100，loss 曲线会怎样？
4. 写出 SGD with Momentum 的更新公式（速度项 + 位置更新）。

## CNN 样例

1. 一个 5x5 输入、3x3 卷积核、stride=1、padding=0，输出尺寸是多少？
2. Max Pooling (2x2, stride=2) 之后，特征图的长和宽各变为原来的多少？
3. 卷积层相比全连接层的两个优势是什么？
4. LeNet-5 用了哪些层类型？按顺序列出来。
