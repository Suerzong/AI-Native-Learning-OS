# 阶段考核

阶段考核用于判断是否形成稳定能力，而不是检查是否背过答案。

## 考核模板

```md
## 阶段名称

- 覆盖技能：
- 考核时间：
- 允许资料：
- 任务：
- 评分标准：
- 晋级条件：
```

---

## 第一层综合考核：神经网络基础

- 覆盖技能：感知机、激活函数、损失函数、前向传播、权重初始化
- 考核时间：60 分钟
- 允许资料：自己的笔记、concepts.md、NumPy 文档
- 任务：
  1. 用 NumPy 从零实现一个 2 层网络的前向传播
     - 输入层：2 个特征
     - 隐藏层：4 个神经元，ReLU 激活
     - 输出层：1 个神经元，Sigmoid 激活
     - 用 Xavier 初始化权重
  2. 计算给定输入的输出值，验证矩阵维度正确
  3. 实现 Binary Cross-Entropy 损失函数
  4. 给定标签，计算 loss 值
  5. 回答：为什么输出层用 Sigmoid + BCE 而不是 ReLU + MSE？
- 评分标准：
  - 代码能运行（30%）
  - 矩阵维度正确（20%）
  - 损失计算正确（20%）
  - 书面回答合理（30%）
- 晋级条件：
  - 代码零错误运行
  - 能解释 Sigmoid + BCE 的选择理由
  - 能手动验算至少一层的输出

## 第二层综合考核：训练原理

- 覆盖技能：反向传播、梯度下降、优化器、学习率
- 考核时间：90 分钟
- 允许资料：自己的笔记、concepts.md
- 任务：
  1. 用 NumPy 从零实现一个完整训练循环：
     - 2 层网络解决 XOR 问题
     - 包含：前向传播、反向传播（链式法则）、参数更新
     - 使用 Adam 优化器
  2. 画出训练过程中 loss 的下降曲线
  3. 调学习率使收敛在 5000 epochs 内完成
  4. 手推：给定一个简单网络，写出从 loss 到 W1 的梯度计算过程
  5. 对比 SGD、Momentum、Adam 在 XOR 上的收敛速度
- 评分标准：
  - XOR 收敛（30%）
  - 反向传播梯度正确（25%）
  - 手推过程正确（25%）
  - 对比实验有据（20%）
- 晋级条件：
  - XOR 4 组输入全部预测正确
  - 手推梯度与代码实现一致
  - 能解释 Adam 为什么比 SGD 收敛更快

## 第三层综合考核：网络架构

- 覆盖技能：CNN、RNN、正则化、数据增强
- 考核时间：120 分钟
- 允许资料：PyTorch 文档、自己的笔记
- 任务：
  1. 用 PyTorch 搭建一个 CNN：
     - 至少 2 个卷积层 + 2 个全连接层
     - 使用 Batch Normalization 和 Dropout
     - 在 CIFAR-10 上训练 20 个 epoch
  2. 目标准确率 > 80%
  3. 画出 train/val 的 loss 和 accuracy 曲线
  4. 回答：
     - 为什么用 3x3 卷积核而不是 5x5？
     - Dropout 的 rate 设为多少？为什么？
     - BN 放在激活函数之前还是之后？为什么？
  5. 尝试数据增强，报告准确率变化
- 评分标准：
  - 模型搭建正确（25%）
  - 准确率达标（25%）
  - 曲线合理（20%）
  - 书面回答有据（20%）
  - 数据增强有效果（10%）
- 晋级条件：
  - 准确率 > 80%
  - 能解释每个架构选择的理由
  - train/val 曲线没有严重过拟合

## 第四层综合考核：实践与端侧准备

- 覆盖技能：NumPy 手写、评估指标、超参数调优、模型压缩、部署
- 考核时间：120 分钟
- 允许资料：PyTorch 文档、ONNX 文档、自己的笔记
- 任务：
  1. 用 NumPy 手写一个 2 层网络并在 XOR 上训练收敛
  2. 用 PyTorch 训练一个分类模型，计算并报告：Accuracy、Precision、Recall、F1
  3. 将模型导出为 ONNX 格式
  4. 写一份部署说明文档，包含：
     - 模型输入输出规格
     - 量化方案建议（INT8 的可行性分析）
     - 目标硬件选型建议
  5. 回答：如果要在 MCU 上跑这个模型，你需要做什么？
- 评分标准：
  - NumPy 实现完整（20%）
  - 评估指标计算正确（20%）
  - ONNX 导出成功（20%）
  - 部署文档合理（25%）
  - 端侧分析有深度（15%）
- 晋级条件：
  - 完整走通训练到导出流程
  - 能解释量化对精度和速度的影响
  - 能说出至少 3 种端侧部署的限制因素
