# 练习库

所有练习按 NNDL 章节编号。格式：`§{章}-{类型}-{序号}`

类型：CONCEPT（概念题）、MATH（数学推导题）、CODE（编程题）、DEBUG（调试题）

出题前必须先判断任务是否属于低价值题。只考四则运算、开方、死记硬背或单纯代公式算数字的题，不应进入本任务库。涉及计算时，除非计算本身能帮助理解核心概念，否则直接给出计算结果，把练习重点放在公式含义、变量解释、参数变化影响、场景选择、错误排查和相近概念区别上。

## 任务模板

```md
## 任务名称

- 价值判断：这道题为什么不是低价值机械题？
- 目标技能：
- 起始材料：
- 任务要求：
- 限制条件：
- 成功标准：
- 常见错误：
- 反馈重点：
```
