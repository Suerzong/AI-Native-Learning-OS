# 诊断量表

每个技能从五个维度诊断。

| 维度 | 诊断问题 | 证据 |
|---|---|---|
| 概念 | 学习者知道这个概念解决什么问题吗？ | 能用自己的话解释 |
| 公式 | 学习者能写出核心公式吗？能解释每个符号的含义吗？ | 能写出公式并解释 |
| 代码 | 学习者能用 NumPy/PyTorch 实现吗？能解释矩阵维度吗？ | 能写出可运行代码 |
| 调试 | 学习者知道 loss 不下降、维度报错时怎么排查吗？ | 能提出排查顺序 |
| 应用 | 学习者知道这个知识点在什么场景下用、和什么搭配吗？ | 能回答适用场景 |

## 诊断输出格式

```md
## 诊断结果

- 当前技能：
- 当前等级：
- 主要弱点：
- 证据：
- 今日训练目标：
- 推荐任务：
```

## 等级判断

| 等级 | 表现 |
|---:|---|
| 0 | 没接触过 |
| 1 | 听过术语，能识别概念 |
| 2 | 能跟着教程写代码，能复述公式 |
| 3 | 能独立写代码实现，能改参数观察变化 |
| 4 | 能迁移到新问题，能解释公式推导 |
| 5 | 能调试异常，能指导别人，能解释设计选择 |
