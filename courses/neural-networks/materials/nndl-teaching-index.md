# NNDL 教学索引

本索引用于约束 `/mastery-learn neural-networks` 的 NNDL-only 教学。它不是新教材，不增加课本外知识；它只规定读取顺序、讲解权重和考察边界。

## 全局规则

- 每个小节都必须读取对应正文。
- 核心知识点重点讲，重要背景轻量讲，了解内容只带过。
- 课本“总结和深入阅读”中的外部书目、论文和工具只作为了解，不进入正式考核。
- 课本习题只有在依赖内容已经讲过时才进入正式练习；涉及后文或外部文献的题先推迟或降级。
- Markdown 标题如果和本索引冲突，以本索引为章节顺序依据，并核对 PDF。

## 题目边界

| 层级 | 可讲 | 可考 |
|---|---|---|
| 核心知识点 | 定义、公式、模型、算法、结构、定理、关键关系 | 概念解释、公式含义、小推导、课本习题 |
| 重要背景 | 历史脉络、问题动机、章节承接、例子背景 | 轻量关系题、动机解释题 |
| 了解 | 人名、年份、工具、参考文献、拓展方向 | 不正式考 |
