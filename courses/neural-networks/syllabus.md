# 神经网络原理与深度学习基础 课程大纲

本课程共 42 节，总时长约 21 小时。面向 Edge AI 方向，按四层递进结构组织。

---

# 课程架构总览

```
第一层：神经网络基础（1-10节）    → 理解神经网络 "是什么" 和 "怎么算"
第二层：训练原理（11-20节）       → 理解神经网络 "怎么学"
第三层：网络架构（21-32节）       → 理解 "有哪些网络" 和 "各自解决什么问题"
第四层：实践与端侧准备（33-42节） → 从 "能跑" 到 "能调" 到 "能部署"
```

---

# 第一层：神经网络基础（第 1-10 节，约 5 小时）

目标：建立神经网络的基本概念，理解数据如何在网络中流动和计算。

| 节号 | 内容 | 核心能力 | 前置依赖 |
|------|------|---------|---------|
| 01 | 什么是神经网络 | 理解神经网络的本质、与传统编程的区别 | 无 |
| 02 | 感知机 | 权重、偏置、激活；线性可分；NumPy 实现感知机逻辑门 | 01 |
| 03 | 线性回归 | MSE 损失；最小二乘法；NumPy 实现 | 02 |
| 04 | 逻辑回归 | Sigmoid；交叉熵损失；二分类；NumPy 实现 | 03 |
| 05 | 激活函数详解 | Sigmoid、Tanh、ReLU、Softmax 的公式、导数和适用场景 | 04 |
| 06 | 损失函数详解 | MSE、Binary CE、Categorical CE 的选择依据 | 05 |
| 07 | 前向传播（单层） | 矩阵乘法、偏置加法、激活函数的向量化实现 | 05、06 |
| 08 | 前向传播（多层） | 多层网络的逐层计算、维度跟踪 | 07 |
| 09 | 权重初始化 | 全零 vs 随机 vs Xavier vs He 初始化 | 08 |
| 10 | 阶段一综合：用 NumPy 实现前向传播 | 综合 2 层网络前向传播 | 08、09 |

## 阶段一掌握标准

用 NumPy 从零实现一个 2 层网络的前向传播：权重初始化、线性变换、激活函数、损失计算。能逐行解释矩阵维度。

# 与 Edge AI 的关联

| 知识点 | Edge AI 关联 |
|--------|-------------|
| 模型压缩 | 嵌入式端算力有限，必须压缩 |
| 量化 | MCU/NPU 通常跑 INT8 推理 |
| 卷积 | CNN 是端侧视觉任务的主力架构 |
| RNN/LSTM | 端侧时序数据处理（传感器融合） |
| 前向传播 | 端侧只做推理不做训练 |
| 参数量计算 | 判断模型是否能放进目标设备 |
