# C++ 注释



程序的注释是解释性语句，您可以在 C++ 代码中包含注释，这将提高源代码的可读性。所有的编程语言都允许某种形式的注释。

C++ 支持单行注释和多行注释。注释中的所有字符会被 C++ 编译器忽略。

C++ 注释一般有两种：

- 

// - 一般用于单行注释。
- 

/* ... */ - 一般用于多行注释。

注释以 // 开始，直到行末为止。例如：
 

## 实例


#include <iostream>
using namespace std;
 
int main() {
  // 这是一个注释
  cout << "Hello World!";
  return 0;
}
